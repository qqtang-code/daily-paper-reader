## 摘要
Transformer与深度状态空间模型（SSM）处于基本设计选择的两端：注意力机制通过基于内容的匹配，以二次成本将每个查询路由至不断增长的键值（KV）缓存中，而深度SSM则将上下文压缩为固定大小的循环状态，该状态无法通过查询-键匹配直接寻址。我们提出跨域注意力（Interdomain Attention），通过核方法将SSM集成到注意力模块中：用一个有限特征映射近似注意力核，将生成的键特征和值投影到由单个SSM循环维护的共享基函数集上，每个查询通过自身的特征映射关注压缩后的系数，从而在固定大小的状态上恢复出由查询条件化的注意力。可扩展层是该推导的一种学习松弛，我们通过消融实验验证其组件。在FineWeb-Edu数据集上进行的125M至1.3B自回归语言建模研究中，在匹配循环状态预算条件下，跨域注意力在各个规模上均优于SSM令牌混合器，在1.3B规模时在验证困惑度和八任务常识推理套件上超越相同配方的softmax基线，并继承了其固定状态核心的长度平坦行为，可外推至训练上下文的3.5倍。消融实验表明，查询条件化投影是增益的主要来源。

## Abstract
Transformers and deep state space models (SSMs) sit at opposite ends of a basic design choice: attention routes each query through a growing key-value (KV) cache by content-based matching at quadratic cost, while deep SSMs compress context into a fixed-size recurrent state that is not directly addressed by query-key matching. We propose Interdomain Attention, which integrates an SSM into an attention module through kernel methods: an attention kernel is approximated by a finite feature map, the resulting key features and values are projected onto a shared set of basis functions maintained by a single SSM recurrence, and each query attends to the compressed coefficients through its own feature map, recovering query-conditioned attention over a fixed-size state. The scalable layer is a learned relaxation of this derivation, and we validate its components through ablations. In a 125M to 1.3B autoregressive language-modeling study on FineWeb-Edu at matched recurrent-state budget, Interdomain Attention improves on an SSM token mixer at every scale, surpasses a same-recipe softmax baseline at 1.3B on validation perplexity and on the eight-task commonsense suite, and inherits the length-flat behavior of its fixed-state core out to 3.5x the training context. Ablations indicate that the query-conditioned projection is the main source of the gain.

### 1. 论文的核心问题与整体含义（研究动机和背景）

Transformer 的 softmax 注意力机制通过逐 token 的键值（KV）缓存实现基于内容的匹配，计算复杂度为 \(O(N^2)\)，且 KV 缓存随序列长度线性增长；而深度状态空间模型（SSM，如 S4、Mamba）将上下文压缩为固定大小的循环状态，每一步更新为 \(O(1)\)，但该状态无法被查询（query）以内容匹配的方式直接寻址。两者各具优势：注意力擅长细粒度检索，SSM 擅长高效长程压缩。现有混合架构通常将两者堆叠，但本文探索一条不同路径：能否让 SSM 在保持固定状态效率的同时，允许每个查询通过内容匹配的方式关注压缩后的历史？该问题构成了本文的核心动机。

**整体含义**：提出一种新的令牌混合器——Interdomain Attention（跨域注意力），将注意力核通过有限特征映射近似，利用 SSM 的循环状态作为共享基函数压缩键和值，使查询能够查询条件化地关注这些压缩系数，从而在固定状态预算下实现线性复杂度，兼顾注意力的灵活性与 SSM 的效率。

### 2. 论文提出的方法论：核心思想、关键技术细节

**核心思想**：从注意力核的 Nadaraya-Watson 核回归视角出发，将 softmax 注意力核替换为可有限特征表示的核（如随机傅里叶特征），然后通过 HiPPO 风格的 SSM 在线维护键特征和值在正交基函数上的投影系数，最终让每个查询通过自身的特征映射与这些压缩系数进行内积，得到条件化输出。

**关键技术细节**：
- **特征映射**：使用 DeltaNet 风格的 SiLU/ℓ2 特征映射 \(\xi(x) = \text{SiLU}(\text{SC}(x)) / \|\text{SiLU}(\text{SC}(x))\|_2\)，其中 SC 为因果深度一维卷积。
- **循环基投影**：采用复对角 S4D 循环（\(x_t = \Lambda \odot x_{t-1} + B z_t\)）模拟 HiPPO 基函数上的投影，将键特征 \(\xi(k_t)\) 与值 \(v_t\) 连接作为 SSM 输入，得到系数矩阵 \(U \in \mathbb{R}^{M \times R}\) 和 \(\Gamma \in \mathbb{R}^{M \times d}\)（\(M\) 为基函数数量，\(R\) 为特征维度）。
- **输出公式**：每个位置的输出近似为 \(\tilde{O} = F_q U^\top \Gamma\)，其中 \(F_q\) 为查询特征矩阵，去掉了归一化分母（参照 GLA、Mamba-2 的无分母线性注意力惯例）。
- **多头结构**：每头独立拥有自己的 S4D 参数（\(\Delta_h, A_h, C_h\)）和投影。
- **可学习松弛**：实际实现并非字面积分，而是使用可学习的 SSM 参数、输入 RMSNorm + bias、无分母读出，将核回归推导作为设计动机，最终层为端到端学习的松弛版本。

**公式（文字说明）**：
$$\tilde{O} = F_q U^\top \Gamma$$
其中 \(F_q = [\xi(q_1); \dots ;\xi(q_{N_q})] \in \mathbb{R}^{N_q \times R}\)，\(U\) 和 \(\Gamma\) 由 S4D 循环更新得到，整个上下文被压缩为固定大小的 \(U, \Gamma\)。

### 6. 论文的主要结论与发现

- **等状态条件下**：Interdomain 在所有规模（125M–1.3B）上均显著优于 S4D-only 基线（验证困惑度降低 13–16%），表明查询条件投影是关键。
- **超越 softmax**：在 1.3B 规模时，Interdomain 在验证困惑度（7.98 vs 8.63）和 8 任务常识推理平均准确率（54.54% vs 51.51%）上超越同配方 softmax；但在小规模（125M）上两者持平。
- **长度外推能力**：Interdomain 继承固定状态核心的长度平坦特性，在 4K 训练上下文下，14K 处困惑度仅变化 ±0.25，而 softmax 暴涨 4.4 倍。
- **消融揭示**：查询条件投影是主要增益来源（单独移除导致困惑度增加 22%），双输入贡献较小（+2.8%），RoPE 正交且在小规模 S4D 家庭中轻微有害。
- **回忆局限性**：Interdomain 在精确字符串检索（如 RULER、Phonebook）上远弱于 softmax，但与 S4D-only 相比仍占优。

### 7. 优点
