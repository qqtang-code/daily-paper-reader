## 摘要
Transformer通过将空间和时间展平为长令牌序列来处理图像和视频。尽管注意力机制和KV缓存保留了过去的特征，但它们的存储随序列长度增长，且缺乏显式、持久的空间状态，这使得长程视频理解和遮挡敏感推理变得困难。我们提出张量记忆（Tensor Memory），这是一个轻量级模块，通过固定大小的循环3D记忆张量增强Transformer块：令牌通过可微软写入操作将内容写入体素网格，该操作将内容以预测连续3D位置周围的高斯加权体积形式沉积；记忆通过高效的局部交互算子和门控循环动态进行更新；令牌通过门控残差融合的连续采样读回上下文。由于记忆张量大小恒定，张量记忆将状态容量与输入长度解耦，同时保留空间归纳偏置。我们在标准语言、图像和视频基准测试以及一个受控的玩具诊断套件上评估该模块，该套件旨在隔离持久状态有益的时机；它与标准Transformer训练流程集成，可以附加到现有块或从现有块中移除，而无需其他架构更改。

## Abstract
Transformers process images and videos by flattening space and time into long token sequences. While attention and KV caching preserve past features, their memory grows with sequence length and they lack an explicit, persistent spatial state, making long-horizon video understanding and occlusion-sensitive reasoning difficult. We propose Tensor Memory, a lightweight module that augments Transformer blocks with a fixed-size recurrent 3D memory tensor: tokens write into a voxel grid via a differentiable soft write that deposits content as a Gaussian-weighted volume around a predicted continuous 3D location, the memory is updated with an efficient local interaction operator and gated recurrent dynamics, and tokens read back context via continuous sampling with gated residual fusion. Because the memory tensor has a constant size, Tensor Memory decouples state capacity from input length while preserving a spatial inductive bias. We evaluate the module on standard language, image, and video benchmarks and on a controlled toy diagnostic suite designed to isolate when persistent state is beneficial; it integrates with standard Transformer training pipelines and can be attached to or removed from existing blocks without other architectural changes.

- **动机**：现有方法虽能扩展上下文长度（如稀疏注意力、压缩记忆、检索增强），但存储的信息通常是碎片的token向量或槽位，缺少保持空间结构的紧凑持久状态。受视觉领域体素网格启发，希望为Transformer提供固定大小、带空间归纳偏置的显式记忆。
- **整体含义**：提出Tensor Memory模块，通过增量写入、局部循环更新和门控读取，使Transformer获得不随输入长度增长的持久3D状态，在语言、图像、视频及诊断任务上验证了有效性。

## 2. 论文提出的方法论
- **核心思想**：在Transformer块内附加一个固定大小的3D体素状态（隐藏状态和细胞状态，形状为 C×D×H×W），令牌分块后依次与记忆交互：写入时预测连续坐标，将特征以高斯加权体积沉积到体素网格；读取时预测查询坐标，通过三线性插值采样记忆并门控融合回令牌流。记忆状态通过因子化的深度可分离3D卷积+ConvLSTM门控循环更新。
- **关键流程**（每块一步）：
  1. **分块**：将令牌序列分组，每块大小为K，块t的读查询为首个令牌 x_read，写信号为整块投影 x_write。
  2. **坐标预测**：共享坐标头（tanh激活）从 x_read 和 x_write 分别预测读坐标 μ_read 和写坐标 μ_write，坐标范围[-1,1]。
  3. **读取**：使用GridSample对当前隐藏状态 h_t 在 μ_read 处三线性插值，得到特征向量，经线性投影到令牌维度，并通过学习标量γ的门控残差（σ(γ)）融合回块内每个令牌。
  4. **写体积生成**：从 x_write 预测内容向量 content 和正展宽 σ，计算高斯掩模 M = exp(-||grid - μ_write||² / (2σ²))，形成写体积 U = content ⊗ M。
  5. **状态更新**：将 U 与 h_t 沿通道拼接，经因子化3D物理算子（三个正交轴向深度可分离卷积+逐点投影）生成四个门控张量，按ConvLSTM规则计算新状态 h_{t+1}, c_{t+1}。
- **可微性与梯度**：高斯写操作避免硬体素赋值，使梯度可传播到写位置和内容；门控残差使记忆路径可被自动抑制。

- **结论**：固定大小的持久3D记忆能有效补充注意力，尤其适合需要维护全局信念的长程、部分观测场景；门控机制可自调节记忆使用。

## 7. 优点
- **架构创新**：将可微连续坐标写入、因子化3D局部更新、门控残差读取有机结合，得到固定大小、强空间偏置的记忆，与Transformer无缝集成。
- **解耦容量与长度**：记忆大小不随输入序列增长，解决了KV缓存线性扩展的问题。
- **可解释性**：可直接可视化写位置、展宽、记忆状态范数、读轨迹、门控值等内部量，无需额外训练目标。
- **模块化**：可插入或移除现有Transformer块而不影响其他部分，学习门控允许任务自适应。
- **诊断设计**：精心设计了4个玩具任务，有效隔离了持久状态的必要性，通过无伤害控制验证了门控的抑制作用。

## 8. 不足与局限

- **记忆容量瓶颈**：固定大小的记忆可能成为存留大量细节的瓶颈，性能可能在记忆分辨率或通道数上饱和。
