## 摘要
视觉-语言-动作策略在长时域和高不确定性控制中仍然脆弱，其中一次性的动作解码提供的推理时推敲有限。显式的思维链可以增加推理深度，但引入了令牌延迟和间接的文本到动作接口。我们提出MPCoT，一种奖励引导的多路径潜在推理框架，初始化M个假设，经过K步权重共享的细化，并在动作解码前进行软聚合。一个仅用于训练的路径偏好目标利用专家动作一致性、基于世界模型/视觉语言模型的进展以及成功反馈来评估候选动作分支，以使潜在路径评分器与下游执行质量对齐。MPCoT保留了原始的8步动作接口，生成零推理令牌，并提供了可配置的推理控制参数(K,M)。在LIBERO和CALVIN上的匹配协议下，MPCoT提升了长时域性能，消融实验证实了深度-宽度效应、置信度加权聚合以及奖励引导的路径监督。

## Abstract
Vision-Language-Action (VLA) policies remain brittle in long-horizon and high-uncertainty control, where one-pass action decoding provides limited inference-time deliberation. Explicit chain-of-thought can increase reasoning depth, but introduces token latency and an indirect text-to-action interface. We propose MPCoT, a reward-guided multi-path latent reasoning framework that initializes $M$ hypotheses, refines them for K weight-tied steps, and softly aggregates them before action decoding. A training-only path-preference objective evaluates candidate action branches with expert-action consistency, world-model/VLM-based progress, and success feedback to align the latent path scorer with downstream execution quality. MPCoT preserves the original 8-step action interface, generates zero reasoning tokens, and exposes configurable inference controls (K,M). Under matched protocols on LIBERO and CALVIN, MPCoT improves long-horizon performance, with ablations confirming depth-width effects, confidence-weighted aggregation, and reward-guided path supervision.

- **核心问题**：视觉-语言-动作（VLA）策略在长时域、高不确定性控制任务中表现脆弱。现有方法多为单次过段动作解码，缺乏推理深度；而显式思维链（CoT）虽能增加推理，但引入文本/视觉 token 延迟，且文本-动作接口不够自然。
- **研究动机**：寻求一种介于浅层单次解码与高代价显式推理之间的折中方案——在连续潜在空间中进行可控的多步、多路径推理，不改变原有动作接口，不产生额外推理 token，从而实现测试时可扩展的计算分配。
- **整体含义**：MPCoT 将 VLA 推理问题重新定义为“体现代测试时计算分配问题”，通过深度（K 步迭代精炼）和宽度（M 条并行假设路径）两个维度来提升长时域决策的鲁棒性。

## 2. 论文提出的方法论
### 核心思想
- 在 VLA 策略的动作解码头前插入一个轻量级潜在推理模块。该模块初始化 **M 条潜在假设路径**，每条路径经过 **K 步权重共享的残差更新**进行精炼，然后通过**可学习的路径评分器**进行软聚合，最后送入不变的动作头解码动作。
- 训练时引入**奖励引导的路径偏好学习**，利用专家动作一致性、世界模型/VLM 评估的进展以及成功信号来监督路径评分器，使其学会选择执行质量更高的分支；推理时移除所有奖励/评估组件，仅使用训练好的评分器进行软聚合。

### 关键技术细节
1. **潜在假设初始化**（式3）：  
   从多模态编码器得到融合上下文 \(c_t\)，通过可学习的假设编码 \(e_m\) 和训练时小噪声 \(\epsilon_{t,m}\) 初始化 M 条路径：  
   \(z_{t,m}^{(0)} = c_t + e_m + \epsilon_{t,m}\)

2. **权重共享多步精炼**（式4-5）：  
   每条路径使用相同的精炼器 \(F_\theta\) 迭代 K 步，每一步预测残差方向并更新：  
   \(r_{t,m}^{(k)} = F_\theta([\text{LN}(z_{t,m}^{(k-1)}); c_t])\)  
   \(z_{t,m}^{(k)} = z_{t,m}^{(k-1)} + r_{t,m}^{(k)}\)

3. **路径评分与软聚合**（式6-7）：  
   轻量级评分器 \(h_\omega\) 输出路径 logits，经 softmax 获得概率 \(p_\omega(m | o_t, l)\)，然后对 M 条路径的最终状态进行加权平均：  
   \(\bar{z}_t^{(K)} = \sum_{m=1}^M p_\omega(m) z_{t,m}^{(K)}\)

4. **奖励引导的路径偏好学习**（训练阶段）：  
   - 每条精炼后的路径解码为候选动作 \(a_{t,m}\)  
   - 计算复合奖励 \(r_{t,m} = \beta_a r_{t,m}^{\text{act}} + \beta_p r_{t,m}^{\text{prog}} + \beta_s r_{t,m}^{\text{succ}}\)，分别对应动作一致性、进展评估、成功标签  
   - 计算路径回报与基线优势，用优势加权交叉熵训练评分器（式A.11）；梯度不通过奖励计算回传

5. **完整损失函数**（式10）：  
   \(\mathcal{L} = \mathcal{L}_{\text{BC}} + \lambda_{\text{RL}} \mathcal{L}_{\text{RL}} + \lambda_{\text{div}} \mathcal{L}_{\text{div}}\)，其中 \(\mathcal{L}_{\text{div}}\) 为多样性正则化项，防止路径坍塌。

### 算法流程（文字说明）
1. 输入观测和指令，经多模态编码器得到融合上下文 \(c_t\)。
2. 初始化 M 条潜在假设路径（式3）。
3. 对每条路径，使用共享精炼器迭代 K 步（式4-5）。
4. 评分器计算每条路径的软权重（式6），并进行加权平均得到聚合潜在表示（式7）。
5. 聚合表示送入不变的动作头，解码出 8 步动作块（式9）。
6. 仅在训练时：每条路径解码候选动作，计算奖励，并更新评分器。

- MPCoT 在 **LIBERO 所有套件**上达到 **98.9% 平均成功率**（单一策略）和 **99.0%**（每套件专用），较 OpenVLA-OFT 显著提升，尤其在 Long 套件（98.9% vs 95.3%）。

## 7. 优点
1. **方法新颖**：在潜在空间进行多路径迭代推理，不产生 token，保持动作接口不变，实现了测试时可控计算分配（K、M 可配置）。
2. **训练-推理分离**：奖励引导路径偏好仅用于训练，推理时无需任何奖励或 VLM 评估，实用性强。
3. **轻量级**：仅增加 2.7% 参数，推理延迟可接受（38ms）。
4. **消融实验全面**：系统验证了深度、宽度、聚合方式、奖励组成等每个组件的贡献，结论可靠。
5. **公平对比**：所有实验在相同骨干和协议下进行，与多个最新方法比较，结果可复现。

## 8. 不足与局限
1. **仅模拟环境验证**：未在真实机器人上部署，无法评估对传感噪声、硬件延迟、接触动力学等真实挑战的鲁棒性。论文承认此局限。
