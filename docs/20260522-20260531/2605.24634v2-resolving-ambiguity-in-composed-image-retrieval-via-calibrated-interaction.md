## 摘要
组合图像检索（CIR）使用参考图像和描述如何修改它的文本来搜索语料库。尽管从三元组训练的合成器到零样本和生成方法取得了快速进展，但所有系统本质上都共享一个假设：查询映射到单个目标，通过Recall@K针对一个标注进行评分。我们认为这与任务根本不一致。像“使其更正式”这样的查询并未命名一张图像，而是语料库的一个区域，用户意图指向哪个成员实际上是不确定的。这种不充分指定是众所周知的假阴性问题的根源，并使当前模型无法区分精确查询和歧义查询。我们将CIR重新定义为不确定性下的校准意图解析：检索器被包裹在一个共形预测层中，该层返回一个具有覆盖保证的候选集，其大小是歧义性的原则性度量；当集合较大时，基于期望信息增益的策略会询问一个来自可解释歧义轴的最有用的澄清问题，并且集合会缩小。我们引入了AmbiCIR，这是一个基准和经过人工验证的用户模拟器，它复兴了CIRR中休眠的辅助和对话标注，并扩展了CIRCO的多正例设置。在开放域和时尚基准上，我们的方法匹配了单轮最先进技术，证实了校准解析在精确查询上无额外成本，同时以远低于朴素对话基线所需的交互预算达到预期目标，并且它是第一个报告该任务有效覆盖和校准的方法。

## Abstract
Composed image retrieval (CIR) searches a corpus with a reference image and a text describing how to modify it. Despite rapid progress from triplet-trained compositors to zero-shot and generative methods, essentially all systems share one assumption: that a query maps to a single target, scored by Recall@K against one annotation. We argue this is fundamentally at odds with the task. A query such as make it more formal does not name an image but a region of the corpus, and which member the user intends is genuinely underdetermined. This underspecification is the root of the well-known false-negative problem and leaves current models unable to tell a precise query from an ambiguous one. We reframe CIR as calibrated intent resolution under uncertainty: a retriever is wrapped in a conformal prediction layer that returns a candidate set with a coverage guarantee and whose size is a principled measure of ambiguity; when the set is large, an expected-information-gain policy asks the single most useful clarifying question, drawn from interpretable ambiguity axes, and the set contracts. We introduce AmbiCIR, a benchmark and human-validated user simulator that revive the dormant auxiliary and dialogue annotations of CIRR and extend the multiple-positive setting of CIRCO. Across open-domain and fashion benchmarks our method matches single-turn state of the art, confirming calibrated resolution is cost-free on precise queries, while reaching the intended target in a fraction of the interaction budget required by naive conversational baselines, and it is the first to report valid coverage and calibration for the task.

## 1. 论文的核心问题与整体含义（研究动机和背景）

论文指出当前组合图像检索（Composed Image Retrieval, CIR）存在根本性假设缺陷：所有现有方法都将查询视为到单一目标的确定性映射，通过Recall@K对单一标注进行评分。然而在实际中，像“使其更正式”这样的查询并不指向唯一图像，而是指定了语料库中的一个区域，用户意图在该区域中实质上是不确定的。这种**查询模糊性**（underspecification）是导致广泛存在的**假阴性问题**（false-negative problem）的根源，使模型无法区分精确查询与模糊查询，输出过度自信的点排序，无法判断查询是否唯一确定答案。

为此，论文重新将CIR定义为**不确定性下的校准意图解析**（calibrated intent resolution under uncertainty）：系统应具备三个能力——（i）量化查询的模糊程度；（ii）在查询精确时，输出带有形式化覆盖保证的候选集；（iii）在查询模糊时，通过提出最优澄清问题逐步消除不确定性。

## 2. 论文提出的方法论：核心思想、关键技术细节

论文的方法围绕三个主要技术组件，且这些组件与底层检索器无关：

- **信念（Belief）生成**：使用冻结的视觉-语言骨干（如BLIP-2、CLIP）加轻量融合适配器，对查询（参考图像+修改文本+可能的对话历史）进行编码，与语料库图像嵌入计算余弦相似度，通过带温度缩放（tempered softmax）得到信念分布 \( p(I|q, h_m) \)。
- **共形预测层（Conformal Prediction Layer）**：采用分割共形预测（split conformal prediction）结合自适应预测集（APS）和Mondrian（组条件）共形预测，在标定集上计算非一致分数经验分位数 \( \hat{\eta}_\alpha \)，从而在测试时构建覆盖保证为 \( 1-\alpha \) 的候选集 \( C_\alpha \)。**候选集大小 \( |C_\alpha| \) 被用作查询模糊性的原则性度量**：集小则查询精确，系统直接提交；集大则触发澄清。
- **信息增益澄清策略**：当系统选择澄清时，从可解释的模糊性轴（保留什么、改变什么、视角、背景）中生成候选问题。利用内部答案模型 \( g(a,I) \) 预测用户答案，计算每个问题 \( a \) 的期望信息增益（Expected Information Gain, EIG），选择信息增益最大的问题提问。用户回答后，通过**语义更新**（将问答对追加到文本侧重新编码）和**逻辑重新加权**（根据软似然 \( \ell(r|a,I) \) 更新信念）联合更新信念，然后重新计算共形集，迭代直到提交或达到最大轮次。
- **训练流程**分三阶段：①训练融合适配器（soft-triplet损失+温度缩放校准）以获得校准的信念；②在标定集上计算Mondrian阈值（无梯度）；③使用策略梯度训练轻量级问题排序器以近似EIG并纠正答案模型偏差。推理算法见于Algorithm 1。

## 6. 论文的主要结论与发现

- **无单轮成本**：校准解析方法在精确查询上匹配甚至略微超过现有最强基线（统计上持平），证明引入不确定性量化不会损害标准检索性能。

## 7. 优点

- **问题定义革新**：首次明确将CIR的模糊性作为核心挑战而非评估噪音，提出“意图解析”新范式，推动领域关注查询不确定性。
- **方法轻量且通用**：共形预测层和信息增益策略均与底层检索器无关，易于集成到任何现代CIR系统。
- **可解释性**：模糊轴（保留、改变、视角、背景）具有场景图级可解释性，问题生成与选择透明。
- **充分的人类验证**：通过30名参与者、600个查询的人类实验验证了模拟器可靠性及用户对提问的满意度，增强了结果可信度。
