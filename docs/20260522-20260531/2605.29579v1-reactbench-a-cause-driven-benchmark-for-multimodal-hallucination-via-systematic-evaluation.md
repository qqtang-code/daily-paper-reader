## 摘要
尽管多模态大语言模型（MLLMs）在视觉-语言理解方面取得了快速进展，它们仍然容易出现多模态幻觉，产生与视觉输入不一致的响应。现有基准主要侧重于检测幻觉结果，而不是评估这些失败的潜在原因。此外，许多基准依赖于简单场景和有限的评估格式，不再能挑战最先进的模型。为了解决这些限制，我们引入了ReactBench，一个因果驱动的幻觉基准，具有多个任务和考试式评估格式。通过生成对抗性图像和诱发幻觉的查询，ReactBench引入了四个目标任务：关系擦除、反事实属性、变更追踪和密集计数。这些任务系统地暴露了共现偏差、语言先验、跨图像比较感知缺陷和细粒度感知瓶颈。除了基于标准精度的评估，我们还利用思维链推理来识别每个任务中幻觉的细粒度子原因。大量评估表明，当前的MLLMs仍然明显容易受到特定原因幻觉触发器的影响，证明了ReactBench作为诊断和提高多模态模型鲁棒性的系统化和可解释测试平台的价值。项目页面位于https://reactbench.github.io/。

## Abstract
While multimodal large language models (MLLMs) have achieved rapid progress in vision-language understanding, they remain prone to multimodal hallucinations, producing responses that are inconsistent with the visual input. Existing benchmarks predominantly focus on detecting hallucination outcomes rather than evaluating the underlying causes of these failures. Moreover, many benchmarks rely on simplistic scenarios and limited evaluation formats that no longer challenge state-of-the-art models. To address these limitations, we introduce ReactBench, a cause-driven hallucination benchmark featuring multiple tasks and an exam-style evaluation format. By generating adversarial images and hallucination-inducing queries, ReactBench introduces four targeted tasks: Relational Erasure, Counterfactual Attribute, Alteration Tracing, and Dense Counting. These tasks systematically expose co-occurrence bias, language priors, cross-image comparative perception deficiencies, and fine-grained perceptual bottlenecks. Beyond standard accuracy-based evaluation, we leverage Chain-of-Thought reasoning to identify fine-grained sub-causes of hallucination within each task. Extensive evaluations reveal that current MLLMs remain notably vulnerable to cause-specific hallucination triggers, demonstrating the value of ReactBench as a systematic and interpretable testbed for diagnosing and improving multimodal model robustness. The project page is available at https://reactbench.github.io/.

# 论文提出的方法论

## 核心思想
- 从四种代表性幻觉原因出发，分别设计四个目标任务：
  1. **关系擦除**：删除高共现对象对中的一个，测试模型是否因共现偏差而幻觉存在该对象。
  2. **反事实属性**：将物体的常见属性改为反常识属性，测试模型是否依赖语言先验而非视觉证据。
  3. **变更追踪**：对图像进行移动、移除、添加、替换等微编辑，测试模型跨图像比较能力。
  4. **密集计数**：选取同一类物体密集出现的图像，测试模型精确计数能力。
- 采用**半自动化流水线**：基于真实图像 → 统计筛选高频率对象对/属性 → MLLM 生成编辑指令 → 图像编辑模型执行编辑 → 生成考试式问答对（MC/SA/YN 三类，含诱导维度）→ 人工审核。
- 引入**思维链（CoT）细粒度子原因分析**：对模型产生幻觉的推理过程进行归因，定义每类任务下的子原因分类，并使用 LLM 自动标注。

## 关键技术细节
- **图像编辑模型**：Qwen-Image-Edit-2511。
- **数据来源**：Visual Genome（前三任务）、FSC147（密集计数）。
- **统计筛选公式**：
  - 共现概率：\( P(B|A) = \frac{C(A \cap B)}{C(A)} \)，条件满足 `C(A∩B) ≥ 50` 且 `max(P(B|A), P(A|B)) ≥ 0.2`。
  - 属性概率：\( P(a|o) = \frac{C(o \cap a)}{C(o)} \)，选取 top-3 作为主导属性，同样阈值筛选。
- **诱导维度设计**：每个任务包含直接提问、常识引导、前提诱导、荒谬对比（部分任务有位置提示/操作提示等）。
- **评估**：使用标准提示和 CoT 提示两种模式，准确率为指标。提出 **React-Score** 层次加权指标：SA 权重 0.5，MC 和 YN 各 0.25；每任务取标准与 CoT 平均值；最终加权（关系擦除和密集计数各 30%，反事实属性和变更追踪各 20%）。

# 论文的主要结论与发现

1. **整体表现受限**：大多数 MLLMs 准确率仅 40%-60%，暗示多模态幻觉仍是未解决挑战。
2. **CoT 效果两极化**：在反事实属性任务中 CoT 能改善（如 +8.0），但在变更追踪和密集计数等感知密集型任务中出现明显下降（最高达 -20.1），表明长推理链可能放大视觉不确定性。
3. **高级模型提升有限**：感知任务（变更追踪、密集计数）上先进模型有优势，但关系擦除和反事实属性任务最高性能仅略超 60%，说明共现偏差和语言先验仍是最顽固问题。
4. **开放式问题最难**：SA 准确率仅 19.1%-28.4%，远低于 MC（50.4%-64.9%）和 YN（58.9%-67.2%）。
5. **子原因分布揭示主要失败模式**：
   - 关系擦除：场景-对象关联（34.8%）和活动-实体完成（33.2%）是主要幻觉来源。
   - 反事实属性：盲目先验断言（49.8%）和常见属性投射（33.3%）主导错误。
   - 变更追踪：跨图像状态同化（47.7%）和细粒度盲点（42.0%）占绝大多数错误。
   - 密集计数：正则捷径错误（49.1%）和顺序枚举缺陷（30.9%）是主要计数失败模式。

# 优点

- **因果驱动框架创新**：从“检测结果”转向“诊断原因”，系统性覆盖四种核心幻觉原因，并提供可解释的归因分析。
- **高质量数据构建**：基于真实图像、统计筛选、MLLM 辅助编辑、人工多重验证，确保视觉保真度和问答有效性。
- **考试式难度梯度**：包含直接到荒谬对比等多种诱导维度，避免简单问题饱和。
- **细粒度 CoT 归因**：首次在每个主要幻觉类别下定义并量化子原因，为模型改进提供具体方向（如增强视觉对比、抑制先验断言）。
- **综合指标 React-Score**：平衡不同难度任务和题型，更全面反映模型能力。

# 不足与局限

1. **覆盖范围局限**：仅针对静态图像中的四种原因，未涵盖视频、多轮对话、音频等更多模态；也未覆盖其他幻觉类型（如关系幻觉、属性幻觉的更多变体）。

4. **未提供缓解方案**：论文聚焦诊断，未提出具体的去幻觉训练或推理技术（仅在 Insights 部分给出方向性建议）。
