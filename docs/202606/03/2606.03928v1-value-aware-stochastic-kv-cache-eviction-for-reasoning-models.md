## 摘要
推理模型通过扩展思维链提高准确性，但其长输出会带来内存和计算瓶颈。KV缓存逐出方法通过从缓存中逐出不重要的键值对来降低开销，但相比保留完整KV缓存、基于选择的稀疏注意力替代方案，其准确性往往更差。我们识别出对KV缓存逐出准确性至关重要的关键因素。首先，一小部分值状态具有异常大的幅值，逐出它们会导致灾难性失败，模型陷入重复推理循环。其次，在逐出过程中引入随机性可通过增加缓存多样性来提高准确性。基于这些发现，我们提出值感知随机KV缓存逐出（VaSE），这是一种无需训练的方法，保护大幅值状态并促进多样化的逐出决策。在六个推理任务上，使用VaSE且KV缓存压缩4倍的Qwen3模型在相同稀疏度下取得的平均准确率高于最先进的（SOTA）选择方法，同时比最强的逐出方法高4%以上。总体而言，VaSE弥合了效率与准确性之间的差距，支持FlashAttention2，并为推理模型实现了静态内存占用。

## Abstract
Reasoning models improve accuracy through extended chains of thought, but their long outputs create a memory and compute bottleneck. KV cache eviction methods reduce this cost by evicting unimportant key-value pairs from the cache, yet they often yield worse accuracy than selection-based sparse attention alternatives, which keep the full KV cache. We identify key factors crucial to KV cache eviction accuracy. First, a small fraction of value states have abnormally large magnitudes, and evicting them causes catastrophic failure where models enter repetitive reasoning loops. Second, introducing stochasticity during eviction improves accuracy by increasing cache diversity. Based on these findings, we propose Value-aware Stochastic KV Cache Eviction (VaSE), a training-free recipe that protects large-magnitude value states and promotes diverse eviction decisions. Across six reasoning tasks, Qwen3 models using VaSE with 4x KV cache compression yield higher average accuracies than SOTA selection method at the same sparsity, while outperforming the strongest eviction method by more than 4%. Overall, VaSE bridges the gap between efficiency and accuracy, supporting FlashAttention2 and enabling a static memory footprint for reasoning models.

# 论文总结：Value-Aware Stochastic KV Cache Eviction for Reasoning Models

## 1. 论文的核心问题与整体含义（研究动机和背景）

*   **核心问题**：推理模型（如 QwQ、DeepSeek-R1 等）通过生成长思维链来提高准确性，但其生成长度可达数万 token，导致 KV 缓存急剧增长，成为内存和计算瓶颈。现有 KV 缓存压缩方法分为两类：
    *   **选择型（selection-based）**：保留完整 KV 缓存，但只激活稀疏子集（如 SeerAttention-R），计算量降低但内存仍线性增长。
    *   **逐出型（eviction-based）**：永久丢弃低重要性 KV 对（如 SnapKV、R-KV、CurDKV），可固定内存占用，但在推理任务上精度明显低于选择型方法。
*   **整体含义**：论文试图弥合逐出型与选择型方法之间的精度差距，使得逐出型方法在保持固定内存优势的同时，达到甚至超越选择型方法的准确性，从而为长上下文推理模型提供高效且实用的部署方案。

## 2. 论文提出的方法论：核心思想、关键技术细节

*   **核心发现 1：大幅值 value states 的关键性**
    *   Value states 的幅值（以 Range(v) = max(v_j) - min(v_j) 度量）分布极度偏斜，少数 token 具有异常大的幅值（outliers）。
    *   实验表明，故意逐出这些大幅值状态会导致灾难性失败——模型陷入重复推理循环，无法产出最终答案，精度从 88.0% 骤降至 14.3%，甚至低于随机逐出（53.2%）。
*   **核心发现 2：随机化的有效性**
    *   在逐出过程中引入随机性（如对注意力分数进行加权采样、重采样随机投影矩阵）可提高缓存多样性，从而提升精度。
    *   例如，SnapKV 用加权采样替换 top-k 选择后精度提升 6.6%；CurDKV 每步重采样高斯投影矩阵后精度提升 9%。
*   **提出方法：VaSE（Value-aware Stochastic Eviction）**，两种变体：
    *   **VaSE-AttnV**：基于 SnapKV，保留顶部 N_v 个最大 Range(v) 的 token 作为保留集 R_V，其余位置从剩余 token 中按注意力分数 ¯α_i 加权采样。
    *   **VaSE-DKV**：基于 CurDKV，每步重新采样随机高斯投影矩阵 G_t，计算乘积得分 ℓ(K)_i * ℓ(V)_i 进行评分与逐出，引入随机性。
    *   **共同关键思想**：在评分函数中引入 value 幅值信号（保护大幅值状态），并引入随机采样机制（提高多样性），两者互补。

*   **VaSE 达到甚至超越选择型方法性能**：在 4× 压缩下，VaSE-AttnV 在 Qwen3-4B 上平均精度 59.09%，超过 SeerAttention-R（58.81%）；在 14B 上 65.81% vs 65.37%。超过最强逐出方法 R-KV 4.4–4.9%。

*   **可推广性**：发现的大幅值价值状态现象与量化误差相关，可启发混合精度量化等未来工作。

## 7. 优点

*   **方法简洁有效**：无需额外训练，仅修改逐出评分和采样策略，即可大幅提升精度，且易集成到现有系统（支持 FlashAttention2）。
*   **发现深度**：首次系统揭示大幅值 value states 在推理模型中的关键作用，并指出随机性能带来缓存多样性增益，具有理论洞察。
*   **实验全面**：覆盖多任务、多模型、多压缩比，做了消融、量化关联、吞吐量/内存基准，对比方法全面，公平性高。
