## 摘要
多模态大语言模型（MLLMs）日益倾向于定位和交织视觉证据以进行审慎推理。基于定位的方法通常通过将裁剪的图像块或特定区域（RoI）特征注入推理上下文来关注兴趣区域（RoIs）。然而，这种设计可能削弱整体场景理解和对象间关系，同时其解码成本随RoI的数量和规模增加而增加。另一种方法是自适应视觉特征选择，但往往需要细粒度的监督或复杂的启发式规则。为解决这些局限，我们提出ROVER（Routing Object-centric Visual Evidence for grounded multi-image Reasoning），一种轻量级、可学习的插件，用于高效的全局视觉证据路由。在每个对象定位预测时，ROVER注入一个步骤特定的令牌三元组，以协同地：(i) 汇总当前推理上下文，(ii) 通过对象中心差异注意力将图像内线索提炼到视觉工作空间，以及(iii) 在此空间内路由并整合跨对象和图像的历史感知证据以支持后续推理。我们将ROVER集成到Qwen2.5-VL-7B中，并开发了交织的SFT-to-GRPO训练流程。严格遵循原始数据集和评估协议，我们的方法在MM-GCoT（答案准确率+4.8%，定位准确率+14.6%）和VideoEspresso（答案准确率+8.6%）上取得了最佳性能。在VideoEspresso上训练的模型展示了强大的迁移能力，在多个基准测试中平均比基础模型高出+4.7%。

## Abstract
Multimodal Large Language Models (MLLMs) have increasingly localized and interleaved visual evidence for deliberative reasoning. Grounding-based approaches typically focus on regions of interest (RoIs) by injecting cropped image patches or RoI-specific features into the reasoning context. However, such designs can weaken holistic scene understanding and inter-object relations, while incurring decoding costs that scale with the number and size of RoIs. Alternatively, adaptive visual feature selection often requires fine-grained supervision or complex heuristics. To address these limitations, we propose ROVER (Routing Object-centric Visual Evidence for grounded multi-image Reasoning), a lightweight, learnable plugin for efficient global visual evidence routing. Upon each object grounding prediction, ROVER injects a step-specific token triplet to synergistically: (i) aggregate the ongoing reasoning context, (ii) distill intra-image cues into a visual working space via object-centric differential attention, and (iii) route and integrate history-aware evidence across objects and images within this space for subsequent reasoning. We integrate ROVER into Qwen2.5-VL-7B and develop an interleaved SFT-to-GRPO training pipeline. Strictly adhering to the original datasets and evaluation protocols, our method achieves the best performance on MM-GCoT (+4.8% answer accuracy, +14.6% grounding accuracy) and VideoEspresso (+8.6% answer accuracy). The VideoEspresso-trained model demonstrates strong transferability, outperforming the base model by +4.7% on average across diverse benchmarks.

## 1. 核心问题与整体含义（研究动机与背景）
- **问题**：多模态大语言模型（MLLMs）在进行接地推理（grounded reasoning）时，现有的两种主流方法存在显著缺陷：
  - **基于兴趣区域（RoI）的注入**：通过裁剪图像块或RoI特征注入推理上下文。这会削弱**整体场景理解**和**对象间关系建模**，同时解码成本随RoI数量和空间尺寸线性增长。
  - **自适应视觉特征选择**：通常需要昂贵的细粒度标注或复杂启发式规则，且主要局限于**单图像场景**，难以扩展到多图像推理。
- **目标**：设计一种轻量、高效、可学习的插件，在保持全局场景理解的前提下，实现跨对象和图像的结构化视觉证据路由，以支持多图像接地推理。

## 2. 方法论：核心思想、关键技术细节、流程
- **核心思想**：受人类视觉认知（选择性注意、工作记忆、返回注视）启发，将每个接地预测扩展为结构化的**证据路由过程**。通过注入恒定长度的token三元组，在语言解码过程中协同执行上下文聚合、图像内线索提纯和历史感知证据整合。
- **关键技术细节**：
  - **触发机制**：当模型预测出有效的接地模式（对象短语+边界框）时，触发路由事件。
  - **路由原语**：在每个接地步骤后立即插入三元组 **Link → Sift → Weave**（共3个token），与推理上下文交织。
    - **Link**：吸收当前推理上下文的紧凑表示。
    - **Sift**：通过**对象中心差分注意力（DiffAttn）**，从非RoI区域提炼互补线索，同时抑制视觉干扰。使用平均池化得到RoI查询向量 \(q_k\)，执行差分交叉注意力：\[\text{DiffAttn}(Q,K,V)=\bigl(\text{softmax}(Q_1K_1^\top/\sqrt{d_h})-\lambda\cdot\text{softmax}(Q_2K_2^\top/\sqrt{d_h})\bigr)V\]当非RoI区域为空时退化为\(q_k\)。
    - **Weave**：通过**历史感知注意力**在**视觉工作空间（VWS）** 上路由并整合之前所有接地对象的信息。VWS是一个按历史顺序累积的矩阵 \(W_k=[W_{k-1}; t_k^{\text{Sift}}]\)，共享于所有图像之间。
  - **解码**：每个接地步骤仅增加3个token，独立于RoI空间尺寸，显著降低解码开销。
- **训练流程**：两阶段交织训练：
  1. **交织SFT**：在教师强制下，接地模式完成后自动插入三元组，用掩码交叉熵损失训练（插入token不计算损失但参与梯度回传）。
  2. **交织GRPO**：对SFT checkpoint进行强化学习优化，以答案准确性为奖励，使用KL散度正则化。

## 6. 主要结论与发现
- **核心发现**：ROVER作为轻量插件，通过结构化对象级证据路由显著提升了多图像接地推理的**准确性**和**解码效率**。
  - MM-GCoT：答案准确率+4.8%（绝对）、接地准确率+14.6%、一致性+12.5%（相对最强基线）。
  - VideoEspresso：平均准确率+8.6%，同时生成token减少31%~58%。
  - **强大迁移性**：仅在VideoEspresso上训练后，在完全不相关的基准上平均提升4.7%，尤其在抗幻觉（HallBench +15.9%）和空间推理（RealWorldQA +5.5%）上表现突出。
- **组件贡献**：
  - 差分注意力是提升接地准确率的关键（比标准注意力高+3.6% G-Acc on VideoEspresso）。
  - VWS和Weave实现跨对象历史整合进一步带来+1.8%收益。
  - GRPO在MM-GCoT上额外提升+4.5% A-Acc。

## 7. 优点：方法或实验设计亮点
1. **轻量高效**：每接地步骤仅增加3个token，额外参数约0.2B（相对于7B骨干），解码成本远低于RoI重新编码/重采样。
2. **全局场景理解**：通过差分注意力从非RoI区域提取互补线索，避免孤立区域导致的上下文缺失和幻觉。
3. **跨图像统一路由**：VWS作为共享工作空间，天然支持多图像中对象证据的历史感知整合，无需显式图像间对齐。
4. **训练友好**：可无缝嵌入任意自回归MLLM的decoder，两阶段SFT+GRPO流程简单有效，不依赖外部工具或复杂启发式。
5. **通用性**：在多个骨干网络（Qwen2.5-VL、Qwen2-VL、InternVL3）上均取得一致收益，零样本迁移性能优异。

## 8. 不足与局限
1. **隐含表示瓶颈**：恒定长度的token压缩可能对极端复杂区域（如密集文本、微小物体）信息容量不足，论文承认需探索自适应分配。
2. **未在更大规模通用基准上验证**：虽然迁移测试覆盖了多个领域，但未在如MMMU、MathVista等综合难度更高的benchmark上报告性能。
3. **视频推理仅用关键帧**：VideoEspresso使用核心帧而非全视频流，可能缺失时序连续性信息。
4. **强化学习仅单数据集**：GRPO仅在MM-GCoT上应用（因VideoEspresso开放性无法设计可靠奖励），整体RL通用性未完全验证。
5. **参数优化空间**：虽只有0.2B额外参数，但进一步压缩至更少（如<0.1B）仍有潜力，论文列为未来方向。
6. **潜在偏差**：训练数据集中于特定领域（MM-GCoT和VideoEspresso），迁移时在部分任务（如MicroVQA提升仅+1.4%）上增益相对较小，表明可能对某些数据类型泛化有限。
