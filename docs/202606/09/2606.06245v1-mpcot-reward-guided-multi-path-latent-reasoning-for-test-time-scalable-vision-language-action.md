## 摘要
视觉-语言-动作(VLA)策略在长期和高不确定性控制中仍然脆弱，其中一次性动作解码提供的推理时推理能力有限。显式的思维链可以增加推理深度，但引入了令牌延迟和间接的文本到动作接口。我们提出MPCoT，一种奖励引导的多路径潜在推理框架，该框架初始化M个假设，通过K个权重共享步骤对其进行优化，并在动作解码前进行软聚合。一个仅训练使用的路径偏好目标通过专家动作一致性、基于世界模型/VLM的进展以及成功反馈来评估候选动作分支，以将潜在路径评分器与下游执行质量对齐。MPCoT保留了原始8步动作接口，生成零推理令牌，并暴露可配置的推理控制(K,M)。在LIBERO和CALVIN的匹配协议下，MPCoT提升了长期性能，消融实验证实了深度-宽度效应、置信度加权聚合以及奖励引导的路径监督。

## Abstract
Vision-Language-Action (VLA) policies remain brittle in long-horizon and high-uncertainty control, where one-pass action decoding provides limited inference-time deliberation. Explicit chain-of-thought can increase reasoning depth, but introduces token latency and an indirect text-to-action interface. We propose MPCoT, a reward-guided multi-path latent reasoning framework that initializes $M$ hypotheses, refines them for K weight-tied steps, and softly aggregates them before action decoding. A training-only path-preference objective evaluates candidate action branches with expert-action consistency, world-model/VLM-based progress, and success feedback to align the latent path scorer with downstream execution quality. MPCoT preserves the original 8-step action interface, generates zero reasoning tokens, and exposes configurable inference controls (K,M). Under matched protocols on LIBERO and CALVIN, MPCoT improves long-horizon performance, with ablations confirming depth-width effects, confidence-weighted aggregation, and reward-guided path supervision.

- **本文目标**：在不改变动作接口、不产生额外推理token的前提下，提供可扩展的推理时计算（test-time compute），通过连续潜在空间的迭代精炼和多路径假设选择来提升长时域决策质量。

## 2. 论文方法论
- **核心思想**：MPCoT（Reward-Guided Multi-Path Latent Reasoning）在现有VLA骨干（OpenVLA-OFT）之前插入一个轻量级潜在推理模块。该模块维护多条潜在路径，通过权重共享的迭代精炼器（weight-tied refiner）重复修正，最后用置信度加权软聚合形成聚合潜在表示，再输入原有策略头（policy head）输出动作块。训练时额外使用路径偏好奖励信号（由专家动作一致性、世界模型/VLM进度、成功反馈组成）来监督潜在路径评分器，使其学会选择执行质量更高的分支。
- **关键技术细节**：
  - **潜在假设初始化**：基于融合上下文 $c_t$ 加上可学习的假设编码 $e_m$ 和小扰动 $\epsilon_{t,m}$ 生成 $M$ 条并行潜在路径起始状态 $z_{t,m}^{(0)}$。
  - **权重共享迭代精炼**：对每条路径，使用相同参数的精炼器 $F_\theta$ 进行 $K$ 步残差更新：
    \[
    r_{t,m}^{(k)} = F_\theta([\mathrm{LN}(z_{t,m}^{(k-1)}); c_t]), \quad z_{t,m}^{(k)} = z_{t,m}^{(k-1)} + r_{t,m}^{(k)}
    \]
    - $F_\theta$ 是轻量MLP，$K$ 控制深度，$M$ 控制宽度。
  - **软聚合**：使用轻量评分器 $h_\omega$ 计算每条路径的得分和软选择概率 $p_\omega(m|o_t,l)$，然后加权平均：
    \[
    \bar{z}_t^{(K)} = \sum_{m=1}^M p_\omega(m|o_t,l) \, z_{t,m}^{(K)}
    \]
  - **训练时路径偏好学习**：将每条精炼路径解码为候选动作块，计算复合奖励 $r_{t,m} = \beta_a r_{t,m}^{act} + \beta_p r_{t,m}^{prog} + \beta_s r_{t,m}^{succ}$，其中动作奖励为与专家动作的$L_2$距离负值，进度奖励由世界模型/VLM评估，成功奖励为二值标签。计算折扣回报和路径平均基线，得到优势 $A_{t,m}$，然后用 advantage-weighted 路径选择损失训练评分器：$\mathcal{L}_{RL} = -\mathbb{E}_{m\sim p_\omega}[\mathrm{sg}(A_{t,m})\log p_\omega(m|o_t,l)]$。推理时完全移除该奖励分支。
  - **总损失**：$\mathcal{L} = \mathcal{L}_{BC} + \lambda_{RL}\mathcal{L}_{RL} + \lambda_{div}\mathcal{L}_{div}$，其中 $\mathcal{L}_{div}$ 是多样性正则项，防止路径坍塌。
- **推理时配置**：$(K,M)$ 可调，如默认 $(5,4)$；不产生推理token，延迟仅线性于 $K\times M$（约31–38ms vs 显式CoT的110–160ms）。
