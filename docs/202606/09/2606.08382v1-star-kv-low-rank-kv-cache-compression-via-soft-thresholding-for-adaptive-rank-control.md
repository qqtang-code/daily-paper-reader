## 摘要
低秩投影通过利用隐藏维度冗余，已成为压缩KV缓存的有前景的方法。然而，现有方法依赖固定或启发式秩选择，难以在最小化精度损失的同时实现激进压缩。我们提出STAR-KV，一种具有细粒度秩控制的自适应低秩KV缓存压缩框架。STAR-KV包含：1）一种可微阈值机制，可在注意力头级别和块级别实现最优秩选择；2）一种混合分解策略，根据键和值投影的敏感性应用不同的低秩分解；3）一种低秩感知的混合精度量化，利用数据统计实现近乎无损的低比特量化。在多个大型语言模型和基准测试上的评估表明，STAR-KV实现了高达75%的KV缓存压缩，结合量化后整体KV缓存减少20倍。通过基于自定义Triton的GPU内核，STAR-KV在注意力模块上实现高达6.9倍加速，端到端生成吞吐量提升3.1倍。我们的代码公开在：https://github.com/PriyanshBhatnagar/STAR-KV。

## Abstract
Low-rank projection has emerged as a promising approach for compressing the KV cache by exploiting hidden-dimension redundancy. However, prior methods rely on fixed or heuristic rank selection and struggle to achieve aggressive compression with minimal accuracy degradation. We propose STAR-KV, an adaptive low-rank KV cache compression framework with fine-grained rank control. STAR-KV encompasses 1) a differentiable thresholding mechanism that enables optimal rank selection at both attention-head and block levels, 2) a hybrid decomposition strategy that applies different low-rank factorizations according to the sensitivity of key and value projections, and 3) a low-rank-aware mixed precision quantization that leverages data statistics for near lossless low-bit quantization. Evaluated across multiple LLMs and benchmarks, STAR-KV achieves up to 75% KV cache compression and up to 20x overall KV cache reduction when combined with quantization. Enabled by custom Triton-based GPU kernels, STAR-KV delivers up to 6.9x speedup for the attention module and 3.1x end-to-end generation throughput. Our code is publicly available at: https://github.com/PriyanshBhatnagar/STAR-KV.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：设计自适应低秩KV缓存压缩框架STAR-KV，通过可微分软阈值机制在注意力头/解码块级别自动学习最优秩，并采用混合分解策略与低秩感知量化进一步压缩。

- **关键技术细节**：

  1. **自适应秩选择（软阈值）**：
     - 对KV投影矩阵执行SVD分解，定义软阈值算子 \( \tilde{\sigma}_i = \sigma_i \cdot \tanh(s(\sigma_i - \alpha)) \)（当\(\sigma_i < \alpha\)时置零），其中\(\alpha\)为可学习阈值，\(s\)控制陡峭度。该算子可微，可通过梯度下降优化。
     - 每个解码块、每个注意力头分别学习\(\alpha\)，从而实现逐块/逐头自适应秩分配。

  2. **混合分解策略**：
     - 联合分解（JD）：将多头投影矩阵拼接后整体SVD，重构误差低但计算开销大（是头分解的h倍）。
     - 头分解（HD）：每头独立分解，重构开销小但误差略高。
     - 基于敏感度分析：键（Key）投影谱衰减更快（低秩误差更小），值（Value）投影更敏感。因此对键用HD（减少开销），对值用JD（降低误差），并通过权重吸收（Weight Absorption）进一步优化值重构的运算顺序。

  3. **低秩感知混合精度量化**：
     - 低秩分解后，潜表示通道按奇异值排序，前20%通道为”异常值“（能量集中），后80%为”正常值“。
     - 对各通道分组（outlier/inlier）分别应用块式Hadamard变换再量化（outlier用4bit，inlier用3bit，平均3.2bit），利用Hadamard平滑分布提升量化鲁棒性。

  4. **训练损失**：

### 7. 优点：方法或实验设计上有哪些亮点

- **自适应秩选择**：首个通过可微软阈值实现逐头/逐层秩学习的框架，自动在块间分配压缩预算，优于静态或启发式分配。

- **低秩感知量化**：利用奇异值排序导致的通道能量差异，设计块式Hadamard+混合精度量化，避免了传统全局Hadamard的精度损失。
- **丰富的消融与跨模型评估**：验证了方法在多个LLM和任务上的泛化性，并系统分析了校准数据集、超参数等敏感性。
