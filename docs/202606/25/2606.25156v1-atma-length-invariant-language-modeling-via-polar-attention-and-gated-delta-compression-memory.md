## 摘要
基于softmax缩放点积注意力的现代大型语言模型受限于其训练序列长度：随着键值序列增长，softmax概率质量会在更广的分布上稀释，导致激活偏移和长上下文性能崩溃。此外，长上下文语言建模面临结构性矛盾：滑动窗口注意力核心保持有界局部表示和低困惑度，但对长距离依赖视而不见；而全上下文注意力保留全局召回，但会遭遇分布外困惑度爆炸。为解决这些限制，我们引入了ATMA，一种混合卷积注意力架构，集成了新型三通道注意力机制。ATMA将注意力混合步骤分解为：(1) 计数无关的单位向量方向通道，(2) 由有效匹配参与比率（经极值校正空汇校正）驱动的有界幅度通道，以及(3) 通过门控增量快速权重规则优化的长期循环压缩记忆。单独来看，极化注意力核心和循环记忆都不足以胜任；它们的组合能够同时实现单调递减的困惑度和高保真长距离检索。我们通过100次全因子消融扫描评估ATMA，证明组合的极化+记忆模型在64K token（训练长度2K的32倍）下仍能保持归纳针束检索准确率高于90%，同时其文档困惑度单调改善，优于在极端上下文长度下崩溃的基于softmax的记忆基线。代码：https://github.com/kreasof-ai/atma

## Abstract
Modern large language models based on softmax scaled-dot-product attention are constrained by their training sequence length: as the key-value sequence grows, softmax probability mass can dilute across a wider distribution, inducing activation shift and long-context performance collapse. Moreover, long-context language modeling faces a structural tension: a sliding-window attention core maintains a bounded local representation and low perplexity but is blind to long-range dependencies, while full-context attention preserves global recall but suffers from out-of-distribution perplexity explosion. To resolve these limitations, we introduce ATMA, a hybrid convolutional-attention architecture that integrates a novel three-channel attention mechanism. ATMA factorizes the attention mixing step into: (1) a count-blind, unit-vector direction channel, (2) a bounded magnitude channel driven by the participation ratio of effective matches over an extreme-value-corrected null sink, and (3) a long-term recurrent compression memory optimized via a gated-delta fast-weights rule. Neither the Polar Attention core nor the recurrent memory is sufficient alone; their combination enables monotonic perplexity reduction and high-fidelity long-range retrieval simultaneously. We evaluate ATMA using a 100-run factorial ablation sweep, demonstrating that the combined Polar + memory model maintains induction needle-in-a-haystack retrieval accuracy above 90% out to 64K tokens (32 times the training length of 2K) while its document perplexity improves monotonically, outperforming softmax-based memory baselines which collapse at extreme context lengths. - **核心问题**：如何构建一种长度不变（length-invariant）的注意力机制，并利用可扩展的记忆模块同时改进困惑度和检索。

## 2. 方法论
- **整体架构**：16层解码器，其中12层为LFM2卷积层（gated convolution），4层为极坐标注意力层（Polar Attention layer）。采用3:1混合比例，预归一化布局（RMSNorm + 残差）。
- **极坐标注意力（Polar Attention）核心思想**：将注意力输出分解为三个正交通道：方向通道（content）、幅度通道（count）和长期记忆通道（memory），输出 = content + count + memory。
  - **方向通道（“什么”）**：对softmax后的值加权和进行L2归一化到单位球面上，消除计数信息，只保留“什么特征被匹配”的方向，实现长度不变。
  - **幅度通道（“多少”）**：计算有效匹配数（基于逆Simpson指数 neff = 1/Σw_hat²），并用sigmoid/ tanh映射到有界区间[0,1)，避免随长度增长而偏移。
  - **极值校正空汇（Null Sink）**：引入可学习的空汇 logit，其值随序列长度对数平方根增长（null = null_base + softplus(γ)·√log(n+1)），防止噪声最大值淹没信号。
  - **长度温度（Length Temperature）**：temp = 1 + softplus(α)·log(n)，随长度增加锐化注意力分布。
  - **辅助干扰损失（Distractor Loss）**：可选的辅助损失，强迫随机键的注意力权重低于空汇，扩大信噪比。
- **门控增量压缩记忆（Gated-Delta Compression Memory）**：基于Titans的“记忆即门”方法，每个头维护一个矩阵状态M ∈ R^{dv×dk}，使用门控增量更新规则：
  - M_t = γ_t·M_{t-1}(I - β_t·k_t k_tᵀ) + β_t·v_t k_tᵀ
  - 关键发现：若键未归一化，特征值γ(1-β||k||²)可能导致发散；通过L2归一化键和查询到单位范数，保证稳定性。
  - 记忆读入：r_t = M_t·q_t，再经RMSNorm和门控后加入残差。
- **算法流程（文字）**：
  1. 对q、k、v应用水平因果卷积（Canon层）和RMS归一化。
  2. 计算原始分数 σ_ij = (q_i·k_j)/√d_k。
  3. 计算长度温度 temp_i 和空汇null_i。
  4. 拼接空汇logit，执行softmax得到权重w。
  5. 计算方向：s = Σ w_j v_j + w_null v_null，再归一化到单位向量c。
  6. 计算幅度：重新归一化w得到w_hat，计算neff，再经tanh映射得到mag。
  7. 组合：out_polar = W_o(reshape(c)·σ(gate)) + W_μ(mag)。
  8. 并行计算记忆输出out_mem，最终输出 = out_polar + out_mem。
- **硬件软件协同设计**：实现了自定义Triton内核（FlashAttention风格前向/反向、GQA分组页式解码、原位门控增量更新），减少HBM流量，提升效率。

## 6. 主要结论与发现
1. **混合架构有效**：仅极坐标注意力或仅记忆都不足以同时保持低困惑度和高检索；两者组合（Polar+记忆）在64K token上检索准确率>90%（长度加权平均94%），且文档困惑度从2.70单调下降至1.96。
2. **softmax+记忆在长上下文检索崩溃**：softmax+记忆配置在64K token时检索准确率仅16%，原因是softmax表示不具备长度不变性，导致记忆读入受到激活漂移干扰。
3. **滑动窗口和干扰损失有害**：当记忆分支激活时，加上滑动窗口会使注意力盲目，干扰损失则过度锐化空汇，两者均降低检索性能。最优配置是无窗口、无干扰损失的全局极坐标注意力+记忆。
4. **记忆自稳定**：门控增量记忆在键L2归一化后状态范数在长度扫描中保持平坦（即使γ=1），无需额外的稳定性措施。
5. **硬件软件协同设计至关重要**：定制Triton内核使记忆分支的MFU下降仅4.5个百分点，同时提升了推理吞吐。
