## 摘要
自主移动机器人在内部物流环境中运行时依赖几何地图进行定位与导航，但缺乏对物体及其上下文属性的语义理解。我们提出了一种上下文语义映射流程，该流程结合了基于SLAM的几何建图、基于SAM的实例分割、实例聚类以及VLM多视角推理，以生成编码几何结构、物体类别和物体可移动性的上下文语义地图表示。通过聚合多视角观测并在零样本、开放词汇设置下查询VLM，该流程推断出上下文物体属性（此处通过可移动性进行演示），无需任务特定训练或预定义物体类别。我们在两种提示策略下评估了三种VLM，并对流程进行了组件分析。所提出的流程在语义分类上达到了98.93%的mIoU，在物体可移动性估计上达到了89.17%的mAcc。组件分析表明，VLM推理是上下文理解的主要瓶颈，而实例聚类是全景性能的主要限制。生成的语义地图支持在动态内部物流环境中进行上下文感知过滤和鲁棒导航。

## Abstract
Autonomous mobile robots operating in intralogistics environments rely on geometric maps for localization and navigation, but lack semantic understanding of objects and their contextual properties. We present a contextual semantic mapping pipeline that combines SLAM-based geometric mapping, SAM-based instance segmentation, instance clustering, and VLM multi-view reasoning to produce a contextual semantic map representation encoding geometric structure, object class, and object movability. By aggregating observations across multiple viewpoints and querying a VLM in a zero-shot, open-vocabulary setting, the pipeline infers contextual object properties--here demonstrated through movability--without requiring task-specific training or predefined object categories. We evaluate three VLMs under two prompting strategies and conduct a component-wise analysis of the pipeline. The proposed pipeline achieves 98.93 % mIoU for semantic classification and 89.17 % mAcc for object movability estimation. Component analysis identifies VLM reasoning as the primary bottleneck for contextual understanding and instance clustering as the main limitation for panoptic performance. The resulting semantic map supports context-aware filtering and robust navigation in dynamic intralogistics environments.

### 1. 论文的核心问题与整体含义（研究动机和背景）

自主移动机器人（AMR）在内部物流环境中广泛使用，依赖 SLAM 生成的几何地图进行定位与导航。然而，这些几何地图仅包含空间结构，缺乏对物体类别、可移动性等上下文语义的理解。这使得机器人难以在动态、异构的物流环境中执行高级任务（如“将托盘从转移站运到货架”）或适应环境变化（如物体移动、状态变更）。现有语义建图方法多基于封闭词汇集的识别模型（如 YOLO），或仅能进行开放词汇的物体识别而缺乏上下文推理。本文旨在填补这一空白：构建一个融合几何建图、实例分割与视觉-语言模型（VLM）多视角推理的上下文语义建图流水线，输出包含物体类别与可移动性属性的语义地图，从而提升 AMR 在动态内部物流中的感知与导航鲁棒性。

### 2. 论文提出的方法论

**核心思想**：将 SLAM 几何建图、SAM 类无关实例分割、实例聚类、VLM 零样本多视角推理整合为模块化流水线。通过多视角聚合观测，利用 VLM 推断物体类别和可移动性（不可移动/可搬运/可自主移动），无需预定义类别或任务特定训练。

**关键技术细节**：
- **传感器配置**：两个 2D 激光扫描仪（360° 测距）和一个前向 RGB 相机，时间同步建立点-像素对应。
- **几何建图**：激光扫描经滤波后，使用 GMapping SLAM 构建 2D 几何地图（点云形式）。
- **实例分割**：对每帧 RGB 图像使用 SAM 自动掩码生成，获得类无关的精细分割掩码，并通过点-像素对应投影到地图坐标系。
- **实例聚类**：基于投影掩码的 IoU（阈值 0.3）将不同帧中同一物体关联起来，形成实例簇，提供持久物体表示并聚合多视角观测。
- **VLM 推理**：对每个实例簇，构建复合图像（全场景遮罩高亮 + 裁剪特写）输入 VLM。使用结构化提示，包含可移动性本体定义和 JSON 输出格式。对比两种提示策略：直接 JSON 输出 和 链式思考（CoT）增强 JSON。VLM 零样本推断物体类别和可移动性，若无法预测则返回未知。
- **地图融合**：将 SLAM 几何点云、实例空间足迹、VLM 预测的类别和可移动性融合，生成最终上下文语义地图（6 维：位置、语义类别、实例 ID、可移动性）。

### 6. 论文的主要结论与发现

- **最佳配置**：Gemini 3.1 Flash Lite 结合直接 JSON 提示在语义分类上最优（98.93% mIoU），同时可移动性估计 mAcc 达 89.17%。GPT-5.1 可移动性 mAcc 更高（96.28%）但语义分类弱（74.38%）。GPT-o3 居中。
- **提示策略影响**：直接 JSON 在语义分类上普遍优于 CoT 增强（CoT 引入不确定性），但可移动性精度相差不大；仅 GPT-o3 受益于 CoT。
- **多视角推理的重要性**：去掉多视图后，语义全景指标大幅下降（mIoU 从 98.93% 降至 65.90%，PQ 从 89.78% 降至 52.83%），而可移动性准确率反而小幅上升（说明单视图偶有正确但整体语义一致性受损）。
- **瓶颈分析**：VLM 推理是可移动性估计的主要瓶颈（替换为真值后 mAcc 提升超 10%）；实例关联（融合阶段）是全景性能的主要限制（替换为真值后 PQ 提升约 9%）；实例分割和聚类不是主要瓶颈；点云-图像融合引入微小误差。

### 7. 优点

- **模块化设计**：各组件（SLAM、SAM、聚类、VLM）可独立替换或升级。
- **零样本开放词汇**：不依赖预定义类别，VLM 可推断未见过物体及其可移动性。
- **多视角聚合**：通过聚类聚合多帧观测，提升推理鲁棒性，克服单视角遮挡与模糊。
- **上下文推理**：不仅识别类别，还能根据情境（如托盘在叉车上视为可移动）调整属性，符合物流真实需求。
- **全面的组件分析**：通过真值替换量化各模块贡献，清晰指出改进方向。
