## 摘要
统一多模态建模旨在单个系统中整合视觉理解与生成。然而，现有方法通常依赖两个不同的视觉分词器，这分割了表示空间并阻碍真正的统一建模。我们提出UniAR，一种统一的自回归框架，其中单个离散视觉分词器作为理解与生成之间的关键桥梁，使得模型能够在无需额外重新编码的情况下直接解释自身生成的视觉标记，从而实现共享上下文。UniAR采用预训练视觉编码器，结合多级特征融合与免查找比特量化方案，在保留高层语义和低层细节的同时，以最小代价扩展有效视觉词汇量。在此基础上，统一自回归模型采用并行比特预测，联合预测空间分组的多级视觉编码，显著缩短视觉序列长度并加速生成。最终，基于扩散的视觉解码器对离散视觉标记进行解码，生成高保真图像。通过大规模预训练，随后进行监督微调和强化学习，UniAR在图像生成和图像编辑任务上达到了最先进的性能，同时在多模态理解基准测试中保持竞争力。项目页面：https://sharelab-sii.github.io/uniar-web。

## Abstract
Unified Multimodal Modeling aims to integrate visual understanding and generation within a single system. However, existing approaches typically rely on two disparate visual tokenizers, which splits the representation space and hinders truly unified modeling. We propose UniAR, a unified autoregressive framework where a single discrete visual tokenizer serves as the key bridge between understanding and generation, enabling a shared context in which the model can directly interpret its own generated visual tokens without additional re-encoding. UniAR adapts a pretrained vision encoder with multi-level feature fusion and a lookup-free bitwise quantization scheme, preserving both high-level semantics and low-level details while scaling the effective visual vocabulary at minimal cost. Building on this, the unified autoregressive model adopts parallel-bitwise-prediction to jointly predict spatially grouped, multi-level visual codes, substantially reducing visual sequence length and accelerating generation. Finally, a diffusion-based visual decoder operates on discrete visual tokens to decode high-fidelity images. Through large-scale pre-training, followed by supervised fine-tuning and reinforcement learning, UniAR achieves state-of-the-art performance on image generation and image editing while remaining competitive on multimodal understanding benchmarks. The project page is available at https://sharelab-sii.github.io/uniar-web.

## 2. 方法论：核心思想、关键技术细节
### 2.1 统一视觉分词器（Unified Visual Tokenizer）
- **多级特征融合**：整合视觉编码器（SigLIP2 ViT）的浅层（保留低层细节）和深层（保留高层语义）特征，通过DeepStack连接，同时满足生成和理解需求。
- **免查找比特量化（BSQ）**：采用Binary Spherical Quantization，将视觉特征映射为二进制向量（如64维，理论词汇量2^64），无需显式码本，显著扩展词汇量且开销低。
- **训练目标**：交叉熵损失（LMM）+ 软熵损失（BSQ），端到端微调视觉编码器，完成后冻结以保证索引一致性。

### 2.2 统一自回归模型
- **预训练LLM**：基于Qwen3-8B，附加4个任务特定Transformer层用于视觉生成。
- **并行比特预测**：在2×2空间网格内同时预测多级BSQ索引，结合空间合并器（Merger）将4个特征合并为1个token，实现32×压缩（512×512图像仅需256个token），生成速度提升4倍。
- **随机索引翻转**：在预训练中随机翻转部分比特，模拟自回归采样误差，增强高温度下的稳定性，对强化学习探索至关重要。
- **损失函数**：文本损失 + λ_vis × 视觉损失，视觉损失直接作用于比特向量。

### 2.3 视觉解码器（Visual Decoder）
- **DiT架构**：基于Stable Diffusion 3.5 Medium DiT，通过元素级加法融合视觉条件信号 f_v（从BSQ索引经特征拼接与投影得到）。
- **分辨率上采样**：对条件信号进行双三次插值，使低分辨率AR预测（如256 token）解码出1024×1024图像，进一步加速。
- **纯视觉驱动**：解码器仅依赖视觉标记，不含文本提示，所有语义和布局生成由AR模型负责。
