## 摘要
我们严格刻画了总参数为$W$、深度为$L$、将长度为$T$的输入序列映射到单个输出的Transformer的VC维，建立了上界$O(L W \log (T W))$和几乎匹配的下界$\Omega(L W \log (T W / L))$。我们进一步严格刻画了使用此类Transformer进行思维链学习的样本复杂度，表明教师强制（即在训练数据上选择与整个思维链一致的预测器）学习的样本复杂度为$O\left(L W \log \left(\left(T+T^{\prime}\right) W\right)\right)$，并且任何使用思维链数据的学习规则至少需要$\Omega\left(L W \log \left(\left(T+T^{\prime}\right) W / L\right)\right)$个样本，其中$T$是输入长度，$T^{\prime}$是自回归步数。

## Abstract
We tightly characterize the VC dimension of depth-$L$ Transformers with a total of $W$ parameters, mapping an input sequence of length $T$ to a single output, establishing an upper bound of $O(L W \log (T W))$ and a nearly matching lower bound of $Ω(L W \log (T W / L))$. We further tightly characterize the sample complexity of chain-of-thought learning using such a Transformer, showing teacher forcing (i.e. selecting a predictor consistent with the entire chain-of-thought on training data) learns with sample complexity $O\left(L W \log \left(\left(T+T^{\prime}\right) W\right)\right)$ and that any learning rule that uses chain-of-thought data requires at least $Ω\left(L W \log \left(\left(T+T^{\prime}\right) W / L\right)\right)$ examples, where $T$ is the input length and $T^{\prime}$ is the number of autoregressive steps.

- **核心思想**：
  - 用 VC 维刻画样本复杂度，无需依赖权重范数，允许任意实数权重。
  - 考虑 hard attention（硬注意力，避免 softmax 的指数函数带来的困难）和 ReLU 激活函数。
  - 通过“分段多项式”框架（Warren 定理）将注意力与 FFN 的离散分支（注意力得分排序、ReLU 激活模式）固定，使输出成为多项式，从而计数符号模式的数量。
- **关键技术细节**：
  - **上界（Theorem 1）**：
    - 将参数空间划分为区域，在每个区域内所有注意力得分符号和 ReLU 激活模式固定。
    - 识别出“分支多项式”：FFN 的预激活多项式（每层 O(NTW) 个）和注意力得分差异多项式（每层 O(N T^3 W) 个）。
    - 应用 Warren 界（多项式符号变化数目上界），得到总区域数乘积界，进而得到 VC 维上界 O(LW log(TW))。
  - **下界（Theorem 2）**：
    - 构造一个“递归检索机器”：利用 T 个上下文位置存储 2^B 种标签配置（B ≈ log T），通过 L 层硬注意力 + FFN 递归地选择位置并读取比特位。
    - 使用 n 组独立样本，每组 L 层 × B 比特，总样本数 nLB ≈ Θ(W L log T)（因为参数 W ≈ Θ(n)）。
    - 同时利用 ReLU 子类的下界 Ω(WL log(W/L))，合并得 Ω(WL log(TW/L))。
  - **CoT 学习（Section 4）**：
    - 定义两个复杂度度量：TSdim（迹打碎维数）控制上界，ASdim（答案打碎维数）控制下界。
    - 上界：将 CoT 损失类（全程轨迹是否匹配）的 VC 维转化为对 Transformer 生长函数的界，通过类似分段多项式方法得到 TSdim = O(WL log((T+T')W))。
    - 下界：通过构造“草稿本”（scratchpad）机制——生成的中间 token 与标签无关（固定前缀），仅在最后一步根据参数读出标签；再利用递归放大，得到 ASdim ≥ Ω(WL log(T+T'))。结合 ReLU 子类下界，最终下界 Ω(WL log((T+T')W/L))/ε。
    - 关键结论：教师强制（Cons_CoT）的样本复杂度达到上述下界，因此是最优的。
