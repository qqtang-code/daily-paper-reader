## 摘要
视频大型多模态模型在视频理解方面取得了显著进展，但它们仍然容易产生幻觉，即生成的响应未得到输入视频的忠实支持。在本文中，我们提出了MultiToP，一种多模态上下文感知的视觉令牌修补框架，通过在语言生成之前细化不可靠的视觉令牌来缓解幻觉。MultiToP引入了一个轻量级的视觉令牌修补器，用于预测令牌级别的替换分布，并选择性地用动态全局修补令牌替换不可靠的视觉令牌。为了有效训练修补器，我们进一步提出了信息引导的排名校准，利用从主干网络获得的答案条件帧级信息线索来指导令牌替换。结合真实答案监督和稀疏正则化，MultiToP实现了无需修改原始模型的局部视觉证据细化。大量实验表明，MultiToP在Vript-HAL上有效减少了幻觉，且推理开销可忽略不计，使得Qwen3-VL-4B-Instruct的F1分数相比原始模型提高了50.60%。同时，MultiToP保留了通用视频理解能力，在Video-LLaVA-7B的ActivityNet-QA上实现了18.58%的相对准确率提升。

## Abstract
Video Large Multimodal Models have achieved remarkable progress in video understanding, yet they remain prone to hallucinations, where generated responses are not faithfully supported by the input video. In this paper, we propose MultiToP, a multimodal-context-aware visual token patching framework that mitigates hallucinations by refining unreliable visual tokens before language generation. MultiToP introduces a lightweight Visual Token Patcher to predict token-level replacement distributions and selectively substitute unreliable visual tokens with a dynamic global patch token. To train the patcher effectively, we further propose information-guided rank calibration, which uses answer-conditioned frame-level information cues derived from the backbone to guide token replacement. Combined with ground-truth answer supervision and sparsity regularization, MultiToP enables localized visual evidence refinement without modifying the original model. Extensive experiments demonstrate that MultiToP effectively reduces hallucinations on Vript-HAL with negligible inference overhead, improving the F1 scores of Qwen3-VL-4B-Instruct by 50.60% over the vanilla model. Meanwhile, MultiToP preserves general video understanding ability, yielding an 18.58% relative accuracy gain on ActivityNet-QA for Video-LLaVA-7B.

## 1. 核心问题与整体含义（研究动机和背景）
视频大型多模态模型（VideoLMMs）在视频理解上表现优异，但生成的回答常与输入视频内容不符，即产生“幻觉”。现有缓解方法主要干预在数据层面（如偏好优化）、视频层面（对比解码）或帧/响应层面（重加权注意机制），但较少关注**视觉令牌**的细粒度可靠性。视觉令牌是语言模型直接消费视频证据的接口，部分令牌可能编码背景噪声、冗余信息或对齐不良，进入语言模型后被错误放大。因此，通过在语言生成前修复不可靠的视觉令牌，有望从根本上缓解幻觉。本文提出 **MultiToP**，一个多模态上下文感知的视觉令牌修补框架，在生成前选择性替换不可靠令牌，以降低幻觉并保持通用视频理解能力。

## 2. 方法论：核心思想、关键技术细节、算法流程
### 核心思想
在语言模型的视觉编码与自回归解码之间插入一个轻量级 **Visual Token Patcher**，该网络根据多模态上下文预测每个视觉令牌是否应被替换，并生成一个动态的全局修补令牌（patch token），用于替代所有被选中的不可靠令牌。

### 关键技术细节
- **令牌替换分布预测**：使用轻量Transformer编码器对多模态令牌序列进行编码，拆分出局部表示和全局上下文，通过MLP输出每个令牌的“保留”/“替换”二分类logits。
- **动态全局修补令牌生成**：将原始视觉令牌均值作为锚点，加上由上下文表示计算得到的残差，得到一个共享的修补令牌，避免为每个被替换令牌生成独立参数。
- **训练框架**（冻结原VideoLMM）：
  1. **可微分替换**：通过Gumbel-Softmax（直通估计器）实现二值替换掩码的可微近似。
  2. **信息引导排名校准**：从原模型中间层收集答案令牌到视觉令牌的注意力分数，聚合为帧级有用信息分数；然后定义帧级替换倾向（logits差值），通过加权排名损失（仅对信息差距大的帧对施加约束）鼓励信息少的帧被更多替换。
  3. **稀疏正则化**：控制总体替换比例ρ，防止过度或不足替换。
  4. **总损失**：`L = L_ce + λ_rank L_rank + λ_sparse L_sparse`，其中L_ce为标准交叉熵。
- **推理阶段**：仅需一次前向传播，无需真实答案或注意力图，引入的额外计算仅为Patcher的前向。

### 算法流程
1. 输入视频经视觉编码器和投影器得到视觉令牌序列v，与问题令牌x拼接。
2. Visual Token Patcher处理多模态序列，输出每个视觉令牌的替换logits和全局修补令牌p。
3. 使用Gumbel-Softmax得到二值掩码g，修补后的视觉令牌 ˜v = (1-g)·v + g·p。
4. 将 ˜v 与x拼接送入冻结的LLM骨干，进行自回归生成。
5. 训练时计算LLM输出的交叉熵、排名损失和稀疏损失；推理时直接输出修补结果。
