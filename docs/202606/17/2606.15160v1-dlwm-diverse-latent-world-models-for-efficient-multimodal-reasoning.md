## 摘要
近年来，多模态大语言模型（MLLMs）的推理能力有了显著提升。现有方法通常依赖于显式的思维链或连续的潜在空间轨迹来增强多步推理。然而，这些方法通常假设输入具有单一的潜在解释，并沿着固定路径或在统一计算预算下展开推理。在真实世界的多模态环境中，视觉观测常常受到遮挡、模糊、视角变化或语义歧义的影响，从而产生多种可能的解释。统一的推理策略不仅限制了模型探索多个假设的能力，还会导致高内存使用和展开成本。我们提出DLWM（多样化潜在世界模型），一种结合潜在空间推理与强化学习的多模态推理框架。首先，我们在连续潜在空间中构建一组多样化的潜在世界假设，每个假设捕捉视觉输入的一种不同合理解释，并在每个假设上独立展开潜在推理。基于正交性的多样性正则化器明确防止假设崩溃。其次，我们将潜在推理过程形式化为资源约束的序贯决策问题，并引入一种资源感知的强化学习策略，该策略自适应地在假设间分配计算资源，动态决定是否扩展、终止或合并推理路径，从而显著降低内存占用并提高展开效率。在多个多模态推理基准上的实验表明，DLWM在准确率上比现有方法高出2-5个百分点，同时内存使用减少24%。

## Abstract
Reasoning capabilities of multimodal large language models (MLLMs) have improved considerably in recent years. Existing approaches typically rely on explicit chain-of-thought or continuous latent-space trajectories to enhance multi-step reasoning. However, these methods generally assume that an input admits a single latent interpretation and unfold reasoning along a fixed path or under a uniform computation budget. In real-world multimodal settings, visual observations are often subject to occlusion, blur, viewpoint variation, or semantic ambiguity, giving rise to multiple plausible interpretations. A uniform reasoning strategy not only limits the model's ability to explore multiple hypotheses but also incurs high memory usage and rollout cost. We present DLWM (Diverse Latent World Models), a multimodal reasoning framework that combines latent-space reasoning with reinforcement learning. First, we construct a set of diverse latent world hypotheses in continuous latent space, each capturing a different plausible interpretation of the visual input, and unfold latent reasoning independently on each hypothesis. An orthogonality-based diversity regularizer explicitly prevents hypothesis collapse. Second, we formulate the latent reasoning process as a resource-constrained sequential decision problem and introduce a resource-aware reinforcement learning policy that adaptively allocates computation across hypotheses, dynamically deciding whether to expand, terminate, or merge reasoning paths, thereby substantially reducing memory footprint and improving rollout efficiency. Experiments on multiple multimodal reasoning benchmarks demonstrate that DLWM outperforms existing methods by 2-5 points in accuracy while reducing memory usage by 24%.

- **核心问题**：现有多模态大语言模型（MLLMs）的推理方法大多假设输入具有单一的潜在解释，并沿着固定路径或在统一计算预算下进行推理。然而，真实世界的视觉输入常因遮挡、模糊、视角变化或语义歧义而存在多种合理诠释，这种单一的推理策略限制了模型探索多种假设的能力，同时导致高内存消耗和低效的计算展开。
- **整体含义**：本文旨在解决两个关键缺陷：①缺乏对多种潜在解释的显式建模，导致在感知不确定性下无法有效进行多假设推理；②缺乏动态计算分配机制，无法根据路径的重要性和任务复杂度自适应调整资源。这两个问题共同制约了多模态推理模型的表达能力和效率。

## 2. 方法论

- **核心思想**：提出 DLWM（Diverse Latent World Models）框架，在连续潜在空间中显式构建多个多样化的潜在世界假设，每个假设对应输入的一种可能的语义解释，并引入资源感知的强化学习策略，自适应地分配计算资源，动态决定是否扩展、终止或合并推理路径，或直接输出答案。
- **关键技术细节**：
  1. **多样化潜在世界假设生成**：
     - 使用多模态编码器（冻结的 ViT-L/14 + Transformer 解码器）得到联合表示 \( z \)。
     - 通过世界生成器 \( G_\phi \)（两层 MLP）结合可学习的世界嵌入 \( e_i \) 生成 \( K \) 个世界表示 \( w_i = G_\phi(z, e_i) \)。
     - 每个世界初始化独立的推理轨迹 \( h_0^{(i)} = w_i \)，由共享的推理模块 \( f_\theta \)（单个 Transformer 块，参数跨世界和时间步共享）迭代更新。
  2. **世界多样性正则化**：采用正交性损失 \( \mathcal{L}_{\text{orth}} = \sum_{i \neq j} \left( \frac{w_i^\top w_j}{\|w_i\|\|w_j\|} \right)^2 \) 惩罚世界表示之间的方向相似度，防止表示坍塌。
  3. **资源感知的强化学习控制器**：
     - 将推理过程建模为预算约束的马可夫决策过程（MDP）：状态 \( s_t = ( \{h_t^{(i)}\}_{i \in A_t}, B_t ) \)，其中 \( A_t \) 是活跃世界集合，\( B_t \) 是剩余预算。
     - 动作空间：Expand(i)、Stop(i)、Merge(i,j)、Answer，分别对应扩展一个世界、终止一个世界、合并两个世界、输出答案。每个动作有不同计算代价（\( c_{\text{exp}}=1.0, c_{\text{stop}}=0.1, c_{\text{merge}}=0.5 \)）。
     - 奖励函数：\( R = R_{\text{acc}} - \lambda_T T - \lambda_K \bar{K} \)，同时鼓励准确率和计算效率。
     - 策略网络 \( \pi_\omega(a|s) \) 通过 REINFORCE 优化，使用移动平均基线降低方差。
  4. **训练流水线**：两阶段训练——先进行固定展开长度的监督预训练，再进行 RL 微调优化计算分配。
  5. **最终预测**：对各活跃世界通过加权融合（可学习评分网络 \( s_\psi \) 分配权重）输出最终答案。

## 6. 主要结论与发现

- DLWM 在所有四个基准上取得最高准确率，优于最强多模态潜在推理基线（LVR、LaRe）2.4–5.2 个百分点，尤其在视觉歧义最强的 MMVP 上提升最大（+5.2）。
- 动态 RL 控制器将平均展开步数从固定 8 步降至 5.0 步，平均活跃世界数仅 2.2，GPU 内存 14.5 GB，与 Heima 效率相当但准确更高。
- 消融证实：多世界建模是核心（K=1 时 MMVP 下降 >5 点）；正交正则化防止坍塌；RL 分配同时提升效率和准确；扩展、停止、合并三个动作互补缺一不可。
- 两阶段训练优于联合训练或单一阶段；加权融合优于平均/最大池化或仅用最后一个世界。
