## 摘要
多模态大语言模型在视觉推理方面表现出色，但依赖于基于文本的思维链，缺乏可解释的视觉中间表示。现有方法使用不透明的标记或外部工具，丢失了关键属性。我们提出Gen-VCoT，一个利用专家视觉模型生成RGB图像作为推理中间表示的框架。它包含三个阶段：视觉定位（SAM分割）、几何推理（Marigold深度图）和语义推理（Qwen2-VL集成）。一个自适应路由器选择推理深度。评估表明，Gen-VCoT在空间问题（提升25%）和深度问题（提升50%）上有所改进，但可能损害简单的事实查询。在CLEVR上，文本思维链优于视觉中间表示（91.2% vs 62.5%），表明最优表示依赖于任务。Gen-VCoT为可解释的多模态推理建立了一个新范式。

## Abstract
Multimodal large language models (MLLMs) excel at visual reasoning but rely on text-based chain-of-thought (CoT), lacking interpretable visual intermediates. Existing methods use opaque tokens or external tools, missing key properties. We propose Gen-VCoT, a framework using expert vision models to generate RGB images as reasoning intermediates. It has three stages: visual grounding (SAM segmentation), geometric reasoning (Marigold depth maps), and semantic reasoning (Qwen2-VL integration). An adaptive router selects reasoning depth. Evaluations show Gen-VCoT improves spatial (25% better) and depth (50% better) questions, but may hurt simple factual queries. Text CoT outperforms visual intermediates on CLEVR (91.2% vs 62.5%), showing task-dependent optimal representations. Gen-VCoT establishes a new paradigm for interpretable multimodal reasoning.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：受到“生成即理解”范式（Vision Banana）启发，将视觉推理分解为三个可解释阶段：视觉定位（Where）、几何推理（How）、语义推理（What），每个阶段生成RGB图像作为中间表示，最终由MLLM综合所有视觉证据给出答案。同时设计自适应推理路由器，根据问题复杂度动态选择推理深度。
- **关键技术细节**：
  - **Stage 1: Visual Grounding（Where）**：使用**SAM ViT-Large**，采用8×8网格点提示（共64个点）生成实例分割图，每个实例赋予不同颜色（高对比度调色板），过滤面积小于100像素的噪声掩码。
  - **Stage 2: Geometric Reasoning（How）**：使用**Marigold LCM**（潜在一致性模型变体，4步推理，5个预测集成）生成密集深度图，再通过彩虹色映射（红→近，紫→远）转换为伪彩色深度图。
  - **Stage 3: Semantic Reasoning（What）**：将原始图像、分割图、深度图三张图片作为多图像输入给**Qwen2-VL-7B-Instruct**，提示模型基于视觉证据回答问题。
  - **自适应推理路由器**：基于BERT-base（1.1亿参数）的分类器，输入问题文本，输出三条路径之一：`path_1`（仅分割）、`path_1+2`（分割+深度）、`full`（完整三阶段）。训练损失包含交叉熵和效率惩罚项（λ=0.1），鼓励选择最简单的充分路径。
- **算法流程**（文字说明）：Router(Q) 选择路径 → 若路径需要分割，则SAM生成并着色分割图 → 若路径需要深度，则Marigold生成并映射深度图 → 根据路径组合将图像输入Qwen2-VL得到答案。
