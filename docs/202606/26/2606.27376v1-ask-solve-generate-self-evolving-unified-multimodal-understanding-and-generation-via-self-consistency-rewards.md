## 摘要
大多数支持视觉理解和图像生成的统一大型多模态模型（LMM）仍依赖于精心设计的后训练监督，例如人工标注、偏好标签或外部奖励模型。我们提出一个问题：统一LMM能否仅使用未标注图像自主提升这两种能力？我们提出一个自我进化的训练框架，包含三个内部角色：Proposer（提案者）生成视觉问题，Solver（求解者）回答问题并评估它们，Generator（生成器）合成图像。训练仅使用自我导出的一致性信号，无需人工标注、偏好标签或任务训练的外部奖励/评判模型。为了稳定学习，我们引入了Solver Token Entropy（STE），一种基于token级预测不确定性的连续难度信号，即使在样本级一致性变得不可靠时仍然有效。对于图像生成，我们设计了一种多尺度内部评估方案，结合了问答忠实度评分和循环一致性描述。这创建了一个求解者中介的耦合，其中更好的视觉理解能够带来更可靠的生成评估和更强的内部训练信号。该框架在基于扩散的BLIP3o、修正流BAGEL和自回归VARGPT-v1.1架构上保持了相同的角色分解、奖励逻辑和训练计划，仅需要每个主干模型的原生提示和生成接口。在八项理解指标上，我们的方法一致地优于相应的基础模型。在BAGEL上，它在MMMU上取得了+3.5%的绝对增益，并将GenEval图像生成性能从82%提升至85%。代码和模型已公开发布。

## Abstract
Most unified large multimodal models (LMMs) that support both visual understanding and image generation still rely on curated post-training supervision, such as human annotations, preference labels, or external reward models. We ask whether a unified LMM can improve both abilities autonomously using only unlabeled images. We propose a self-evolving training framework with three internal roles: a Proposer that generates visual questions, a Solver that answers and evaluates them, and a Generator that synthesizes images. Training uses only self-derived consistency signals, without human annotations, preference labels, or task-trained external reward/judge models. To stabilize learning, we introduce Solver Token Entropy (STE), a continuous difficulty signal based on token-level prediction uncertainty that remains useful even when sample-level consistency becomes unreliable. For image generation, we design a multi-scale internal evaluation scheme that combines question-answer fidelity scoring with cycle-consistent captioning. This creates a solver-mediated coupling, where better visual understanding enables more reliable generation assessment and stronger internal training signals. The framework preserves the same role decomposition, reward logic, and training schedule across diffusion-based BLIP3o, rectified-flow BAGEL, and autoregressive VARGPT-v1.1 architectures, requiring only each backbone's native prompting and generation interface. Across eight understanding metrics, our method consistently improves over the corresponding base models. On BAGEL, it achieves a $+3.5\%$ absolute gain on MMMU and improves GenEval image generation performance from $82\%$ to $85\%$. Code and models are publicly released.

### 论文提出的方法论

**核心思想**：将统一模型分解为三个角色适配器（LoRA），并在冻结主干上通过内部自一致性信号训练。训练由两个循环构成：理解循环（Proposer-Solver）和生成循环（Generator+作为评估者的Solver）。核心是通过**求解者中介耦合**，让理解能力的提升自动改善生成侧评估信号。

**关键技术细节**：
- **Proposer**：生成与图像相关的视觉问题，目标是将问题定位到Solver的“能力前沿”（既非太简单也非无法回答）。
- **Solver**：回答Proposer提出的问题，并在多个提示变异下评估答案一致性。
- **Generator**：根据文本提示生成图像，并由Solver进行评估。
- **奖励信号**：理解奖励结合自一致性熵与**求解者Token熵(STE)**——STE基于token级预测最大熵，经滚动窗归一化，能有效避免样本级一致性退化（即所有提示给出相同答案但模型实际不确定）。生成奖励则通过**QA忠实度评分**（比较生成图像与真实图像上对同一组诊断问题的答案匹配）与**循环一致性描述**（生成图像的描述与原始提示的语义相似度）综合计算。

**算法流程（文字说明）**：
1. 使用LoRA在冻结主干上初始化三个角色适配器。
2. 对于每张训练图像：
   - 理解步：Proposer采样K个问题，Solver在N种提示框架下回答，计算自一致性熵和STE，构造奖励Rp和Rs，用GRPO和REINFORCE分别更新Proposer和Solver。
   - 生成步：Solver对真实图像生成描述作为提示，Generator合成L个候选图像，Solver对每个候选图像执行QA忠实度和循环一致性评估，计算奖励Rg，用主干原生生成目标（自回归的token-policy或扩散/流的奖励加权去噪）更新Generator。
3. 两个循环按3:2比例交替，全部训练信号仅源自内部。

### 优点

1. **完全自监督**：无需任何人工标注、偏好标签或外部奖励/评判模型，仅用未标注图像产生训练信号。
2. **高通用性**：角色分解、奖励定义、训练计划在所有主干间固定，仅需主干原生界面，即可推广至扩散、修正流、自回归三种主流生成范式。
3. **创新技术点**：提出STE解决样本级一致性退化；设计QA忠实度和循环一致性结合的生成评估；实现求解者中介非对称耦合。

### 不足与局限

1. **数据规模有限**：仅使用10,000张未标记图像，更大规模的数据和更长训练步数可能带来更多提升，但也会显著增加算力开销（论文提到3.2k GPU小时，对大规模实验可能不够）。
2. **依赖Solver质量**：Solver既是学习者又是评估者，若初始理解能力较弱，可能产生低质量评估信号，限制生成端增益。尤其对初始能力很差的小模型可能不适用。
3. **全参数微调不稳定**：实验表明全微调反而不如LoRA，说明内部奖励下适配器更新更安全，但也可能限制了模型上限。
4. **生成端困难子类提升有限**：如VARGPT-v1.1上的“位置”和“颜色属性”仅从13%、21%提升到15%、24%，说明自回归架构的根本限制无法被语义级监督完全克服。
5. **未探索更大规模或更强主干**：仅测试了7B~8B规模的模型，未在更大模型（如数十B）上验证泛化性。
6. **未讨论安全与对齐风险**：自进化训练可能放大固有偏见或产生未预期行为，论文未涉及。
