## 摘要
网络架构设计的最新进展引入了层注意力以增强层间交互。在此类框架中，每一层查询所有前置层以建立跨层连接。然而，层注意力导致计算复杂度随网络深度呈二次增长。为解决此问题，先前工作提出了循环层注意力（RLA）和线性注意力机制，但这些方法存在静态信息更新和长距离跨层依赖建模能力有限的问题。为克服这些限制，我们提出关键相关层注意力（KCLA），灵感来自我们对层注意力中关键表示具有高余弦相似度的观察。KCLA 实现了线性计算复杂度，同时保持了动态信息更新，直接源自层注意力的基础定义。此外，KCLA 保持了长距离跨层连接，并具有固定空间复杂度，不依赖于网络深度。实验评估表明，KCLA 在包括图像识别、目标检测和医学图像分割在内的多种任务中均表现出良好性能。代码已公开于 https://github.com/bgx666/KCLA。

## Abstract
Recent advances in network architecture design have introduced layer attention to enhance inter-layer interactions. In such frameworks, each layer queries all preceding layers to establish cross-layer connections. However, layer attention results in quadratic computational complexity with respect to network depth. To mitigate this issue, prior works have proposed Recurrent Layer Attention (RLA) and linear attention mechanisms, which suffer from static information updates and limited long-range cross-layer dependency modeling. To overcome these limitations, we propose Key-Correlated Layer Attention (KCLA), inspired by our observation that Key representations in layer attention exhibit high cosine similarity. KCLA achieves linear computational complexity while preserving dynamic information updates, directly derived from the foundational definition of layer attention. Furthermore, KCLA maintains long-range cross-layer connections and features a fixed spatial complexity, independent of network depth. Empirical evaluations demonstrate that KCLA delivers good performance across diverse tasks, including image recognition, object detection, and medical image segmentation. The code is publicly available at https://github.com/bgx666/KCLA.

### 1. 核心问题与整体含义（研究动机和背景）
- **背景**：层注意力机制（Layer Attention）将每一层的输出视为序列中的token，通过查询所有前置层来增强跨层交互。但标准层注意力计算复杂度随网络深度 `D` 呈二次增长 `O(D²)`，并且需要缓存所有前置层的Key/Value，导致显存开销也随深度线性增长。
- **现有方法的局限**：
  - **循环层注意力（RLA）**：通过缓存前一时刻的Key/Value并拼接当前层投影来降低重复计算，但Key/Value一旦生成便不再更新，成为静态机制，限制了信息动态性。
  - **轻量版MRLA-L**：用线性近似将注意力输出分解为递归形式，但导致早期层的贡献呈指数衰减，无法建立有效的长距离跨层依赖。
  - **动态层注意力（DLA）**：引入额外动态共享单元更新表示，参数容量增大，且未解决复杂度问题。
- **核心问题**：如何设计一种线性复杂度、固定空间开销、保留动态信息更新且能建模长程依赖的层注意力机制。

### 2. 方法论：核心思想与关键细节
**核心思想**：基于经验观察——标准层注意力中不同层之间的Key表示方向高度相关（余弦相似度高）。利用这一性质，可以直接从层注意力的定义出发进行线性近似，避免静态问题。

**关键技术细节**：
- **线性近似推导**：忽略softmax后，层注意力输出可写为  
  `O_t = Σ_{i=1}^t Q_t (K_{t,i,:})^T V_{t,i,:}`  
  利用Key方向相关性，近似为  
  `O_t ≈ ||Q_t|| cosθ f_t^V( Σ_{i=1}^t ||K_{t,i,:}|| X_i )`  
  其中 `cosθ` 为统一夹角，`f_t^V` 为线性变换。通过多粒度（Multi-Head Attention）降低单头维度，使线性近似更准确。

- **递归形式**：定义 `U_t = U_{t-1} + ||K_{t,i,:}|| X_t`，实现 `O(t)` 复杂度更新，无需缓存所有前置层输出。

- **恢复softmax**：引入指数变换与head-wise温度参数。生成对数均匀序列 `P`（范围 `(p, 1/p)`），为每个head设置不同缩放因子 `D_h`。通过线性层 `Φ` 将 `||Q_t|| cosθ` 映射为head注意力分数，实现动态温度调节，从而动态调整注意力分布。

- **跨阶段设计**：处理不同stage之间通道数和空间尺寸变化。通过1×1卷积和自适应池化对累积状态 `U_t` 和归一化因子 `z_t` 进行适配（算法1）。
