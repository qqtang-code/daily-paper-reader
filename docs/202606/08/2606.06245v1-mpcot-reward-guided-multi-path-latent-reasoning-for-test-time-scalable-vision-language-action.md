## 摘要
视觉-语言-动作（VLA）策略在长周期和高不确定性控制中仍然脆弱，其中单次动作解码提供的推理时推断有限。显式思维链可以增加推理深度，但引入了令牌延迟和间接的文本到动作接口。我们提出MPCoT，一种奖励引导的多路径潜在推理框架，初始化M个假设，对它们进行K次权重共享步骤的细化，并在动作解码前进行软聚合。一个仅训练阶段的路径偏好目标使用专家动作一致性、基于世界模型/VLM的进展以及成功反馈来评估候选动作分支，从而将潜在路径评分器与下游执行质量对齐。MPCoT保留了原始的8步动作接口，生成零推理令牌，并提供了可配置的推理控制（K, M）。在LIBERO和CALVIN上的匹配协议下，MPCoT提高了长周期性能，消融实验证实了深度-宽度效应、置信加权聚合以及奖励引导的路径监督。

## Abstract
Vision-Language-Action (VLA) policies remain brittle in long-horizon and high-uncertainty control, where one-pass action decoding provides limited inference-time deliberation. Explicit chain-of-thought can increase reasoning depth, but introduces token latency and an indirect text-to-action interface. We propose MPCoT, a reward-guided multi-path latent reasoning framework that initializes $M$ hypotheses, refines them for K weight-tied steps, and softly aggregates them before action decoding. A training-only path-preference objective evaluates candidate action branches with expert-action consistency, world-model/VLM-based progress, and success feedback to align the latent path scorer with downstream execution quality. MPCoT preserves the original 8-step action interface, generates zero reasoning tokens, and exposes configurable inference controls (K,M). Under matched protocols on LIBERO and CALVIN, MPCoT improves long-horizon performance, with ablations confirming depth-width effects, confidence-weighted aggregation, and reward-guided path supervision.

视觉-语言-动作（VLA）策略在长周期（long-horizon）和高不确定性控制任务中表现脆弱，其核心问题在于**单次动作解码缺乏推理时（inference-time）的深思熟虑**。显式思维链（CoT）虽能增加推理深度，但引入令牌延迟和间接的文本-动作接口，效率低下。MPCoT旨在通过**潜在空间（latent space）中的多路径、多步迭代推理**来弥补这一不足，在不改变原有动作接口、不生成推理令牌的前提下，提升VLA模型在长时域任务中的执行稳定性与成功率。

### 2. 提出的方法论

- **核心思想**：在动作解码前，利用一个轻量的潜在推理模块对同一个感知-语言上下文的多个假设进行迭代细化和软聚合，从而在不产生显式推理文本的情况下实现“思考”。
- **关键技术细节**：
  - **多路径初始化**：由多模态融合得到的上下文向量 \(c_t\) 加上可学习的路径编码 \(e_m\) 和训练时的小扰动，生成 \(M\) 条并行的潜在假设 \(z^{(0)}_{t,m}\)。
  - **权重共享的多步细化**：使用同一个共享的细化算子 \(F_\theta\) 对每条路径进行 \(K\) 步残差更新（公式 4-5），每一步都重新与上下文 \(c_t\) 条件化，防止指令漂移。

  - **奖励引导的路径偏好学习（训练专用）**：在训练时，将每条路径解码为候选动作，并使用专家动作一致性、世界模型/VLM 的进展评估以及成功标签计算组合奖励 \(r_{t,m}\)（公式 8），然后通过优势加权路径选择损失 \(L_{RL}\) 训练评分器。推理时完全去除该奖励分支，评分器充当执行质量的代理。

- **公式与算法流程**：
  1. 编码输入得到上下文 \(c_t\)。
  2. 生成 \(M\) 个初始潜在向量 \(z^{(0)}_{t,m}\)。
  3. 对每个 \(m\) 迭代 \(k=1..K\)：\(z^{(k)}_{t,m} = z^{(k-1)}_{t,m} + F_\theta([\text{LN}(z^{(k-1)}_{t,m}); c_t])\)。

### 6. 主要结论与发现

- MPCoT 在 LIBERO 全部套件上平均 SR 达到 98.9%（优于 OpenVLA-OFT 的 96.8%），长序列 SR 从 95.3% 提升至 98.9%。
- 在 CALVIN ABC→D 上，5步 SR 达到 89.4%，比 OpenVLA-OFT 提高 16.5 个百分点，增益随任务长度增大。
- 深度 K 和宽度 M 存在互补效应：多步细化提升单路径质量，多路径提供替代假设，两者结合最佳。
- 奖励引导的路径偏好显著提升路径一致性（Path Consistency 从 68.5% 升至 84.3%）和最终任务成功率，且复合奖励优于单一奖励。
- MPCoT 在推理时不引入推理令牌，延迟仅从 24ms（直接解码）增至 38ms（K=5,M=4），远低于显式 CoT 的 110-160ms。

### 7. 优点

- **接口友好**：保持原有 8 步动作接口，不改变动作解码器，可即插即用到现有 VLA 骨干。
- **零推理令牌**：所有中间推理在连续潜在空间完成，避免文本令牌的存储和推理开销。
- **可配置计算**：通过参数 (K, M) 灵活控制推理时计算量，支持测试时可扩展（test-time scaling）。
- **训练-推理分离**：奖励引导的评分器仅在训练时使用奖励和世界模型，推理时完全无需奖励/卷动，保持高效。
- **设计模块化**：潜在推理模块与骨干解耦，易于与其他 VLA 改进（如更好的动作表征）正交结合。

### 8. 不足与局限

- **仅在模拟环境中验证**：结果来自 LIBERO 和 CALVIN 模拟器，未在真实机器人上测试，缺乏对传感器噪声、硬件延迟、接触动力学等现实因素的鲁棒性分析。
- **依赖训练时的外部评估器**：路径偏好学习需要一个训练专用（training-only）的 VLM/世界模型进展评估器，该评估器的质量会影响训练效果，且对评估器的依赖增加了系统复杂度。
- **潜在过拟合风险**：在 CALVIN 等特定基准上大幅提升，可能部分归因于对基准任务结构的过拟合，跨领域泛化能力需进一步验证。
- **缺乏真实世界演示**：论文指出孤立真实机器人演示也可能具有任务/平台特异性，但作为补充验证仍不足以完全证明实用性。
