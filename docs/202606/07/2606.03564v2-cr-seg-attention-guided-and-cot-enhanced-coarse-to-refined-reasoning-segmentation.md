## 摘要
推理分割旨在通过联合视觉-文本推理，对复杂语言描述的目标对象进行分割。现有方法通常要么依赖学习到的语义标记来桥接多模态大语言模型（MLLMs）与分割模型，面临跨模态对齐困难的问题；要么使用显式空间提示（如边界框），这可能会丢失整体的响应语义。针对这些局限，我们提出注意力引导与思维链增强的由粗到精推理分割，称为CR-Seg，一个用于由粗到精推理分割的两阶段框架。具体地，我们设计了提取注意力图与点（EAP）模块，用于提取注意力图进行粗略目标定位，并选择信息点，两者均输入SAM进行掩码细化。为了缓解推理与答案不一致问题，我们进一步引入全局到局部思维链（GLCoT），引导模型从全局场景上下文逐步推理到局部目标细节。在推理分割基准上的大量实验证明了CR-Seg的有效性。

## Abstract
Reasoning segmentation aims to segment target objects described by complex language through joint visual-textual reasoning. Existing methods typically rely on either learned semantic tokens to bridge Multimodal Large Language Models (MLLMs) and segmentation models, suffering from difficult cross-modal alignment, or explicit spatial prompts such as bounding boxes, which may lose holistic response semantics. To address these limitations, we propose Attention-Guided and CoT-Enhanced Coarse-to-Refined Reasoning Segmentation, termed CR-Seg, a two-stage framework for coarse-to-refined reasoning segmentation. Specifically, we design an Extract Attention Maps and Points (EAP) module to extract attention maps for coarse target localization and select informative points, both of which are fed into SAM for mask refinement. To alleviate reasoning--answer inconsistency, we further introduce Global-to-Local Chain-of-Thought (GLCoT), which guides the model to reason progressively from global scene context to local target details. Extensive experiments on reasoning segmentation benchmarks demonstrate the effectiveness of CR-Seg.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：推理分割（Reasoning Segmentation）任务要求模型能根据复杂语言描述（如“下雨天用于挡雨的工具”）对目标对象进行分割。现有方法分为两类：
  - **内部表示法（IRS）**：使用 MLLM 的隐藏状态（如 [SEG] token）作为分割接口，但缺乏显式空间监督，跨模态对齐困难。
  - **显式提示法（ERS）**：训练 MLLM 输出边框或点等显式空间提示，但将完整推理过程压缩为稀疏提示，丢失整体语义，且分割质量高度依赖定位准确性。
- **现状**：两种范式在“语义完整性”和“对齐难度”上各有缺陷；且现有模型容易产生**推理–答案不一致**（reasoning–answer inconsistency），即推理过程中被干扰项误导，导致错误定位和掩码。
- **动机**：需要一种既能保留 MLLM 整体响应语义、又可显式监督、且兼容 SAM 细化的“空间-语义桥梁”。注意力图（attention maps）具有密集空间分布、可掩码级监督的特点，是理想候选。

## 2. 方法论（核心思想、关键技术细节）

### 2.1 总体框架：两阶段训练（粗→精）
- **推理模型**：使用 Qwen3-VL-4B 作为 MLLM，生成结构化输出（`<thinking>` 推理 + `<answer>` 标签和边框）。
- **分割模型**：使用 SAM3，接收掩码先验（mask prior）、正/负点提示，产出最终掩码。
- **核心创新**：
  1. **EAP 模块**：从 MLLM 注意力层提取原始 logits（跳过 Softmax 避免过度平滑），保留视觉 token 部分，聚合得到注意力图 Att；再从 Att 中按累积注意力质量选择正/负点。
  2. **GLCoT 思维链**：三步渐进推理（全局场景分析→中粒定位→细粒定位），蒸馏自大模型 Qwen3-VL-235B，用于缓解推理-答案不一致。

### 2.2 EAP 模块（提取注意力图与点）
1. **注意力提取**：在 MLLM 响应末尾追加可学习查询（learnable queries），通过前向传播获得其输出嵌入，再提取对应层的原始 QK^T 矩阵，保留视觉 token 部分，得到 `A' ∈ R^(L×V×N)`。
2. **聚合与缩放**：对层和查询维度求平均，缩放到 SAM 输入分辨率（H×W），得到 `Att`。
3. **掩码先验生成**：通过四层卷积轻量网络（Connector）将 Att 投影到 SAM 输入空间，得到 `MP`。
4. **点采样**：
   - 正点：从 Att 值降序排列，取累积注意力超过 `α_fg * total` 的前缀区域，选最高点 + 4 个按概率采样点。
   - 负点：从升序排列中取累积超过 `α_bg * total` 的区域，均匀采样 5 个点。
   - 第二训练阶段因目标小，正点减少为 1 个。

### 2.3 损失函数
- **阶段 1（预训练对齐）**：对齐损失 `Lali = λ1 Lcos + λ2 Lseg`
  - `Lcos`：余弦相似度（间接监督注意力图与真实掩码的对齐）
  - `Lseg`：分割损失 = DICE + BCE + 边界损失（Sobel 边缘 L1 差异）
  - 冻结图像编码器，仅微调 MLLM 的 LoRA（q,k,v,o）及 SAM 的 Prompt Encoder 和 Mask Decoder。
- **阶段 2（监督微调）**：联合优化生成损失 `LSFT`（交叉熵）和对齐损失 `Lali`，用 GLCoT 数据集训练。

1. **CR-Seg 在 FReasonSeg 上达到 SOTA**，尤其在 L2/L3 子集中表现突出，证明注意力先验对同类别干扰具有鲁棒性。

4. **注意力桥接的优势**：相比 IRS 减少对齐训练量，相比 ERS 保留整体响应语义；即使喂入错误推理，注意力仍能关注正确区域（图5）。
5. **可扩展性**：更大模型（8B）或更强骨干（SAM3）可进一步提升性能。

## 7. 优点（方法与实验亮点）

- **创新性**：首次将 MLLM 注意力图用作可显式监督的“空间-语义桥”，兼具 IRS 的语义完整性和 ERS 的显式监督能力。
- **EAP 设计巧妙**：通过跳过 Softmax、聚合层和查询维度、阈值采样点，将注意力图转化为 SAM 兼容的掩码先验和点提示，无需修改 SAM 架构。
- **GLCoT 有效性强**：三步渐进推理机制简单实用，不仅能提升分割性能，还显著改善逻辑一致性。
- **新基准 FReasonSeg**：填补现有基准缺乏同类别细粒度区分的空白，提供更全面的评估维度。
- **实验全面且公平**：覆盖多种基线、消融、缩放实验，且复现基线在相同条件下比较，消融细致（组件、查询数、思维链、骨干）。

## 8. 不足与局限

- **依赖 SAM 架构**：注意力图与 SAM 原生输入格式不完全匹配，限制了进一步性能提升。未来可探索更简化的细化策略。
- **仍需额外训练**：虽然比 IRS 对齐更高效，但仍需要两阶段训练以及 LoRA 微调，并非完全即插即用。
- **FReasonSeg 规模较小**（仅 283 样本），且由大模型自动生成 + 人工审核，可能存在噪声或偏差。
- **未探讨多目标/复杂场景**：当前主要集中在单目标或少量目标，多目标联合分割的泛化性未充分验证。
- **推理开销**：GLCoT 需额外生成三步推理，增加响应长度（139 词 vs 无 CoT 的 93 词），虽性能提升但推理延迟增加。
- **公平性潜在偏差**：使用的 teacher 模型（Qwen3-VL-235B）可能存在偏见，间接影响 GLCoT 数据质量。
