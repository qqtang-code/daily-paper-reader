## 摘要
多模态大语言模型（MLLMs）在视觉推理方面表现出色，但依赖基于文本的思维链（CoT），缺乏可解释的视觉中间表示。现有方法使用不透明的标记或外部工具，缺失关键特性。我们提出Gen-VCoT框架，利用专家视觉模型生成RGB图像作为推理中间表示。该框架包含三个阶段：视觉定位（SAM分割）、几何推理（Marigold深度图）和语义推理（Qwen2-VL集成）。自适应路由器选择推理深度。评估显示，Gen-VCoT在空间问题（提升25%）和深度问题（提升50%）上有改善，但可能降低简单事实查询的性能。文本CoT在CLEVR上优于视觉中间表示（91.2% vs 62.5%），表明最优表示依赖于任务。Gen-VCoT为可解释的多模态推理建立了新范式。

## Abstract
Multimodal large language models (MLLMs) excel at visual reasoning but rely on text-based chain-of-thought (CoT), lacking interpretable visual intermediates. Existing methods use opaque tokens or external tools, missing key properties. We propose Gen-VCoT, a framework using expert vision models to generate RGB images as reasoning intermediates. It has three stages: visual grounding (SAM segmentation), geometric reasoning (Marigold depth maps), and semantic reasoning (Qwen2-VL integration). An adaptive router selects reasoning depth. Evaluations show Gen-VCoT improves spatial (25% better) and depth (50% better) questions, but may hurt simple factual queries. Text CoT outperforms visual intermediates on CLEVR (91.2% vs 62.5%), showing task-dependent optimal representations. Gen-VCoT establishes a new paradigm for interpretable multimodal reasoning.

- **问题**：多模态大语言模型（MLLMs）在视觉推理中主要依赖基于文本的思维链（CoT），这种语言媒介对于空间推理是间接且低效的。现有视觉CoT方法要么使用不透明的连续视觉标记（如COVT），要么依赖外部工具调用（如Visual Sketchpad），无法同时实现**可解释性**、**端到端训练**和**密集视觉表示**三个关键特性。
- **背景**：人类在视觉推理时会自然地进行绘制、标注和心智操作（如画框、突出区域、估计深度），而当前MLLM缺乏外化中间视觉推理产品的能力。论文受“图像生成模型是通用视觉学习者”（Vision Banana）的启发，认为生成分割图或深度图的过程本身就编码了场景理解所需的视觉知识。
- **核心含义**：首次系统性地使用专家视觉模型生成的RGB图像作为视觉推理的中间表示，建立“生成即理解”的可解释多模态推理新范式。

## 2. 方法论：核心思想、关键技术细节、流程

### 核心思想
- 将视觉推理分解为三个阶段，每个阶段输出RGB图像作为可解释的中间表示：
  1. **视觉定位（Where）**：通过SAM生成实例分割图，突出与问题相关的区域。
  2. **几何推理（How）**：通过Marigold生成伪彩色深度图，建立空间深度关系。
  3. **语义推理（What）**：Qwen2-VL整合原始图像、分割图和深度图，生成最终答案。
- 设计**自适应推理路由器**，根据问题复杂度动态选择推理深度（仅分割、分割+深度、全流水线或基线）。

### 关键技术细节
- **Stage 1：视觉定位**：使用SAM ViT-Large，以8×8网格点提示（64个点）覆盖全图，生成实例掩码；过滤面积<100像素的噪声掩码；为每个实例分配高对比度颜色。
- **Stage 2：几何推理**：使用Marigold LCM（4步推理，5次集成）生成深度图；归一化后映射到彩虹colormap（红色=近，紫色=远）。
- **Stage 3：语义推理**：Qwen2-VL-7B-Instruct接收三张图（原始+分割+深度）和问题，输出答案。提示模板明确告知三图的含义。
- **自适应路由器**：基于BERT-base（110M参数）的分类器，输入问题文本，输出三种路径（path_1、path_1+2、full）。训练损失为交叉熵 + λ·step_count（鼓励选择最简单路径），λ=0.1。

### 算法流程（文字描述）
1. 路由器根据问题文本选择推理路径。
2. 若路径包含分割：SAM生成并着色分割图。
3. 若路径包含深度：Marigold生成伪彩色深度图。
4. 将原始图、分割图、深度图（根据路径）传入Qwen2-VL，生成答案。

## 6. 主要结论与发现

- **Gen-VCoT提升空间与深度推理**：在复杂场景上，空间问题准确率提升25%（从3/4到4/4），深度问题提升50%（从2/4到3/4），总准确率提升10.5%（从68.4%到78.9%）。
- **简单事实查询下降**：在CLEVR上，Gen-VCoT准确率仅70.0%，低于基线的85.0%。视觉中间表示引入噪声（SAM多余掩码、Marigold无效深度）。
- **文本CoT更优**：在CLEVR上文本CoT达91.2%，远超Gen-VCoT（62.5%）和基线（85.0%），表明结构化文本对事实查询是更精确的通道。
- **最优表示任务依赖**：空间/深度问题视觉中间表示有效，事实/属性问题文本描述更优。这需路由器能选择视觉或文本中间表示（未来方向）。
- **分割与深度互补**：消融证实全流水线优于任何部分配置，分割帮助计数和识别，深度帮助空间排序。

- **自适应路由设计**：根据问题复杂度动态选择推理深度，平衡效率与准确性，具有实用价值。
- **分析全面**：不仅报告准确率，还分析了何时视觉中间表示有帮助、何时有害，并以文本CoT作为对比，揭示了任务依赖的特性。
