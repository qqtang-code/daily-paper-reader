## 摘要
ABACUS是一个统一的视觉语言模型，无需任何基准特定训练即可处理物体计数、人群计数、指代表达式计数以及忠实于计数的图像生成。我们的模型构建于现有的30亿参数统一基础模型之上，并通过三项关键创新适配于目标定位任务：利用目标性图进行密度感知自适应缩放以实现空间定位；通过GRPO的边界感知计数策略消除裁剪边界误差；以及采用循环一致性GRPO策略，其中理解分支自我批评生成的输出，无需任何外部标注即可弥合理解与生成之间的差距。ABACUS在七个基准测试中取得了最先进的结果，超越了任务特定专家模型和更大的通用模型。

## Abstract
ABACUS is a unified vision-language model that handles object counting, crowd counting, referring-expression counting, and count-faithful image generation without any benchmark-specific training required. Our model is built on existing 3B-parameter unified foundation model and is adapted for object localization tasks using three key innovations: density-aware adaptive zooming with objectness maps for spatial grounding; a boundary-aware count policy via GRPO to eliminate crop-boundary errors; and a cycle-consistent GRPO strategy where the understanding branch self-critiques generated outputs, closing the understanding-generation gap without any external annotations. ABACUS achieves state-of-the-art results across seven benchmarks, outperforming both task-specific specialists and larger generalist models.

- **核心问题**：现有视觉模型在物体计数、人群计数、指代表达式计数等**计数理解任务**与“计数忠实图像生成”（即生成指定数量对象的图像）之间存在割裂。任务专用模型性能优异但缺乏泛化性，统一多模态模型（如UniLIP）虽能同时处理两类任务，却存在**协同差距（synergy gap）**：同一模型正确计数4个苹果，却无法生成恰好4个苹果的图像。此外，MLLM在稠密场景中计数精度差，易产生粗粒度估计（如“大于100”）。
- **研究动机**：构建一个**单一统一模型**，同时实现高精度的计数理解（物体/人群/指代计数）和计数忠实图像生成，且无需针对每个基准进行额外训练，缩小理解与生成之间的差距。
- **整体含义**：证明计数理解与生成是**相互促进**的任务，联合优化可以产生涌现的空间感知能力，为多模态基础模型在空间推理任务中的应用提供新范式。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

**核心思想**：基于UniLIP-3B统一基础模型，通过三项创新适配计数任务：①密度感知自适应缩放与对象性图增强空间定位；②边界感知计数策略消除裁剪误差；③循环一致性GRPO策略让理解分支自批评生成输出，无外部监督地弥合理解-生成差距。

### 关键技术细节（文字说明）

#### a) 密度感知自适应缩放（Density-aware Adaptive Zooming）
- 使用GroundingDINO + MLP头构建**密度指示器** \(\phi(\cdot)\)，判断区域为稀疏或稠密。
- 稠密区域递归切分为2×2子区域（最小分辨率\(\gamma\)），使MLLM在局部稀疏区域计数更可靠。
- 各子区域独立计数后聚合为全局计数。

#### b) 对象性图注入（Objectness Map）
- 从MLLM注意力层提取多头自注意力分解，经结构掩码和共享仿射对齐得到每个视觉token的**对象性得分** \(o_{l,i}(v)\)。
- 计算交叉注意力权重分布 \(q_l(v)\)，通过**对象性正则化损失** \(L_{obj}\) 监督（使用高斯平滑真值点标注），迫使模型内化实例级空间结构，避免虚假标记记忆。

#### c) 边界感知计数策略（Boundary-aware Count Policy）
- 针对自适应缩放造成的切割边界物体归属问题，训练MLLM输出结构化输出：将物体分类为**内部（interior）**、**边缘（edge）**、**边界（boundary）**，每象限内部计数 \(n_q^{\text{int}}\)、边缘计数 \(c_q^e\)、边界计数 \(d_q^e\)，总计数公式：
  \[
  s_q = n_q^{\text{int}} + \sum_{e\in E_q} c_q^e,\quad \hat{T} = \sum_{q\in Q} s_q
  \]
- 通过GRPO后训练，使用**三重嵌套奖励**（局部精度 \(\Delta_q^r\)、边界一致性 \(\Delta_b^r\)、全局一致性 \(\Delta_g^r\)），奖励函数为：
  \[
  \Delta_r = \exp\left(-\frac{|\text{pred} - \text{GT}|}{\text{GT} + \epsilon}\right)
  \]
- 组相对优势计算：\(A_i = (R(y_i) - \mu_R)/(\sigma_R + \epsilon)\)，优化目标：\(L_{\text{GRPO}}(\theta) = E[\sum_i A_i \log \pi_\theta(y_i|x)] - \beta D_{KL}(\pi_\theta \| \pi_{\text{ref}})\)

#### d) 循环一致性GRPO生成增强（Cycle-consistent GRPO）
- 生成分支根据计数条件提示采样K个候选图像，**冻结的理解分支**对每个候选图像计数，计算**计数偏差奖励**：
  \[
  R_{\text{cnt}}(\hat{x}_i) = \exp\left(-\frac{|\hat{c}_i - c^*|}{c^* + \epsilon}\right)
  \]
- 结合美学得分 \(S_{\text{aes}}\) 形成复合奖励：\(R = \lambda_c R_{\text{cnt}} + \lambda_a S_{\text{aes}}\)。
- 仅更新生成分支LoRA，理解分支冻结，避免奖励信号退化。循环训练中生成分支逐渐产生更逼真的多实例场景，进一步锐化理解分支的奖励信号。

### 训练策略（三阶段）

1. **Stage 1：理解分支训练**  
   在2M稠密标注图像上使用 \(L_{\text{und}} = L_{\text{SFT}} + L_{\text{obj}}\) 微调MLLM（LoRA），后接边界感知GRPO（50K样本）。
2. **Stage 2：连接器训练**

3. **Stage 3：生成分支训练**
