## 摘要
多模态大语言模型（MLLMs）在视觉推理方面表现出色，但依赖于基于文本的思维链（CoT），缺乏可解释的视觉中间变量。现有方法使用不透明的token或外部工具，缺失关键属性。我们提出Gen-VCoT，一个使用专家视觉模型生成RGB图像作为推理中间变量的框架。它包含三个阶段：视觉定位（SAM分割）、几何推理（Marigold深度图）和语义推理（Qwen2-VL集成）。自适应路由器选择推理深度。评估显示，Gen-VCoT在空间问题（提升25%）和深度问题（提升50%）上表现更好，但可能损害简单事实查询。在CLEVR上，文本CoT优于视觉中间变量（91.2% vs 62.5%），表明任务依赖的最优表示。Gen-VCoT为可解释的多模态推理建立了新范式。

## Abstract
Multimodal large language models (MLLMs) excel at visual reasoning but rely on text-based chain-of-thought (CoT), lacking interpretable visual intermediates. Existing methods use opaque tokens or external tools, missing key properties. We propose Gen-VCoT, a framework using expert vision models to generate RGB images as reasoning intermediates. It has three stages: visual grounding (SAM segmentation), geometric reasoning (Marigold depth maps), and semantic reasoning (Qwen2-VL integration). An adaptive router selects reasoning depth. Evaluations show Gen-VCoT improves spatial (25% better) and depth (50% better) questions, but may hurt simple factual queries. Text CoT outperforms visual intermediates on CLEVR (91.2% vs 62.5%), showing task-dependent optimal representations. Gen-VCoT establishes a new paradigm for interpretable multimodal reasoning.

- **研究动机**：多模态大语言模型（MLLMs）虽在视觉推理上表现优异，但其推理过程主要依赖文本链式思维（CoT），**缺乏可解释的中间视觉表示**。人类在视觉推理中会自然地画出边界框、高亮区域、估计深度，而MLLMs没有这种外化中间产品的能力。

- **本文贡献**：提出Gen-VCoT，**第一篇系统利用专家视觉模型生成RGB图像作为视觉推理中间步骤**的框架，为可解释多模态推理建立新范式。

### 2. 方法论

- **核心思想**：受Vision Banana等启发，将视觉理解视为“生成即理解”——生成分割图或深度图的过程本身编码了场景理解所需知识。Gen-VCoT将推理分解为三个可解释阶段。
- **三阶段流水线**：
  - **Stage 1 – 视觉定位（Where）**：使用SAM（ViT-Large）在8×8网格点提示下生成实例分割图，每个实例赋予高对比度颜色。
  - **Stage 2 – 几何推理（How）**：使用Marigold（LCM变体，4步去噪，5个预测集成）估计深度图，并映射为伪彩色（红色=近，紫色=远）。
  - **Stage 3 – 语义推理（What）**：Qwen2-VL-7B-Instruct同时接收原始图像、分割图和深度图，依据Prompt“基于这三个视觉证据简洁回答……”生成最终答案。
- **自适应推理路由器**：一个BERT-base分类器（110M参数），根据问题文本预测三条路径之一：
  - `path_1`：仅使用分割（适用于物体级别问题）
  - `path_1+2`：分割+深度（适用于空间问题）
  - `full`：三阶段全部（适用于复杂推理）
- **训练目标**：交叉熵 + λ·step_count 惩罚，鼓励选择最简单路径。
- **算法流程**（Algorithm 1）：先通过路由器选路径，按需生成中间图像，最后由Qwen2-VL推理。
