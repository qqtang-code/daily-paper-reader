## 摘要
同策略自蒸馏（OPSD）在其自身的推出上训练模型，并使用冻结副本来提供以参考目标为条件的密集词元级目标。这对于LLM推理效果很好，但直接扩展到多模态大语言模型（MLLMs）可能会产生一种捷径：特权目标可能主要基于文本参考目标而非图像来引导词元。我们提出ViGOS，一个面向MLLM后训练的视觉接地OPSD框架。学生首先编写视觉描述，然后推理得到最终答案。对于有效的推出，仅图像的感知教师监督描述，而特权推理教师监督同一学生前缀上的推理和最终答案。参考教师仅用于无效推出以恢复输出格式。在通用视觉-语言、专家推理、视觉数学、空间接地和视觉-语言先验基准测试中，ViGOS保持了OPSD的主要优势，并在易受捷径影响的设置中改进了基于图像的行为。

## Abstract
On-policy self-distillation (OPSD) trains a model on its own rollouts and uses a frozen copy to provide dense token-level targets conditioned on a reference target. This works well for LLM reasoning, but a direct extension to multimodal large language models (MLLMs) can create a shortcut: the privileged target may guide tokens mainly based on the text reference target rather than the image. We propose ViGOS, a visually grounded OPSD framework for MLLM post-training. The student first writes a visual description and then reasons toward the final answer. For valid rollouts, an image-only perception teacher supervises the description, while a privileged reasoning teacher supervises the reasoning and final answer on the same student prefix. A reference teacher is used only for invalid rollouts to recover the output format. Across general vision-language, expert reasoning, visual math, spatial grounding, and visual-language-prior benchmarks, ViGOS keeps the main benefits of OPSD and improves image-grounded behavior in shortcut-prone settings.

## 1. 核心问题与整体含义（研究动机和背景）
- **背景**：同策略自蒸馏（OPSD）在纯文本 LLM 推理中效果显著，通过冻结的教师模型对学生自身的推出（rollout）提供密集词元级监督，并结合参考答案进行指导。
- **问题**：直接将 OPSD 扩展到多模态大语言模型（MLLM）会导致一种文本捷径：特权教师能同时看到图像、问题和参考答案，而学生仅看到图像和问题。由于 MLLM 更容易被文本信号主导，教师可能**主要依据文本参考目标**而非图像内容来修正学生词元，从而促使学生学到“答案兼容但视觉接地弱”的推理路径，引发类似于 VQA 中语言先验和捷径学习的问题。
- **整体含义**：需要一种方法，在保留 OPSD 密集监督优势的同时，**强制模型在利用答案指导推理之前，先充分感知和描述图像内容**，从而提升多模态推理的鲁棒性和图像接地性。

## 2. 方法论：ViGOS 框架
- **核心思想**：将感知（视觉描述）与推理（答案推导）解耦，在训练时对不同阶段使用不同上下文条件的教师进行监督，切断参考答案对早期视觉描述的直接影响。
- **关键技术细节**：
  - **结构化输出格式**：学生生成形如 `<description> d </description> <think> r </think> \boxed{a}` 的序列，包含视觉描述 `d`、推理过程 `r`、最终答案 `a`。
  - **教师角色（均为冻结的学生副本）**：  
    - **仅图像感知教师 (q_img)**：仅接收图像，用于监督描述部分 `d`，强制描述依赖视觉证据而非文本提示或答案。  
    - **特权推理教师 (q_rea)**：接收图像、问题、参考答案，用于监督推理和答案部分 (`r`, `a`)，在学生已写出描述的基础上提供答案导向的推理指导。  
    - **参考教师 (q_ref)**：接收全量上下文，仅当推出格式无效（如缺少标签）时使用反向 KL 散度恢复格式。
  - **损失函数**：
    - 有效推出：`L_ViGOS = λ_perc L_perc + λ_rea L_rea`，其中 `L_perc` 为感知教师与学生的 KL（对描述词元），`L_rea` 为推理教师与学生的 KL（对推理+答案词元）。
    - 无效推出：`L_ref` 为学生与参考教师的反向 KL（对整个推出）。
    - 所有教师均在**学生自身的前缀**上计算分布，保持同策略性。
- **算法流程**（文字）：
  1. 从学生采样结构化推出。  
  2. 解析器提取描述词元集 Td、推理词元集 Tr、答案词元集 Ta，并判断有效/无效。  
  3. 有效时，分别用 q_img 和 q_rea 计算相应片段的 KL 损失；无效时用 q_ref 计算反向 KL 损失。  
  4. 汇总损失更新学生参数（教师参数冻结）。  
  5. 推理时仅使用学生，无需教师或参考答案。

## 6. 主要结论与发现
- **ViGOS 保持了 OPSD 的主要基准增益**：在两个骨干上，Pass@5 和 Avg@5 均优于 Baseline，与 OPSD 接近或略优。  
- **ViGOS 显著改善了先验敏感场景下的图像接地性**：在 ViLP 上得分更高，同时保持较高的 Prior 准确率（不盲目抑制先验）。  
- **PALR 诊断证实设计目标**：ViGOS 在推理-答案片段上的答案驱动修正比例显著低于 OPSD（3B: 6.33% vs 17.26%；7B: 7.56% vs 26.01%），描述片段几乎为零。  
- **消融实验表明三个组件均必要**：移除感知损失导致 ViLP 和 CV-Bench 下降；移除推理损失导致总体下降；移除参考教师或改用前向 KL 均导致性能降低。  
- **训练动力学**：OPSD 的 ViLP 得分先升后降，而 ViGOS 持续上升，说明分离监督有助于稳定利用图像。

## 7. 优点
- **方法简洁有效**：通过结构化输出和分段教师实现感知与推理的显式解耦，不增加额外标注成本，推理时无需教师。  
- **诊断工具 PALR 新颖**：提供了一种定量衡量答案/图像驱动贡献的方法，可推广到其他多模态训练方法分析。  
- **实验设计严谨**：包含同提示控制、消融、训练动力学、定性分析，全面验证方法有效性。  
- **应用价值高**：针对 OPSD 在多模态场景下的实际缺陷提出修复，适用于需要强图像接地性的视觉问答、数学推理等任务。

## 8. 不足与局限
- **实验覆盖不足**：仅在 Qwen2.5-VL 系列上验证，未在 LLaVA、InternVL 等其他主流 MLLM 上测试，泛化性未知。  
- **训练资源消耗**：需要额外 3 个教师前向传播（每次推出额外 3 倍计算），虽可通过缓存优化，但开销仍比标准 OPSD 大。  
- **依赖格式解析**：结构化输出可能因生成错误而失效，参考教师虽能恢复但引入额外信号。  
- **描述质量不稳定**：仅图像教师可能产生过于通用或不准确的描述，学生可能学习不精确的描述策略。  
- **基准局限性**：ViLP 仅测试特定类型的先验冲突，其他形式捷径（如选项分布、数据集偏差）未被评估。  
- **答案泄漏度量本身有假设**：PALR 基于单一图像和答案反事实，并非完全归因，可能遗漏其他影响因素（如文本选项、语言模型先验）。  
- **未报告推理时间开销**：结构化输出会增加生成长度，可能影响部署延迟。
