## 摘要
我们提出Echo-Infinity，一个面向实时无限视频生成的自回归（AR）框架，它采用可学习的演化记忆，以恒定代价动态过滤、抽象和压缩任意长度的历史信息。现有方法主要通过预定义的KV缓存调度、固定比例启发式压缩或推理时RoPE适配来管理记忆。这些设计由于缓存窗口有限且忽视自回归生成噪声，不可避免地丢失历史信息并放大累积误差。受人类记忆巩固过程的启发，Echo-Infinity用可学习的记忆查询（Memory Query）替换了手工设计的记忆管理，当过去帧从局部窗口被驱逐时，这些查询通过注意力机制和门控机制进行更新。这些查询与视频扩散变换器（DiTs）进行端到端联合优化，形成一种演化记忆，支持任意压缩比，计算量恒定且与视频长度无关。它们还充当可泛化的生成先验，即使仅使用优化后的初始状态也能提升生成质量。我们进一步引入了统一相对RoPE配方（Unified Relative RoPE Recipe），它将沉降帧锚定在id 0开始，并在整个训练和推理过程中让最新帧id最多增长到DiTs预训练的最大时间RoPE id，从而将模型从有限RoPE约束中解放出来，并弥合了训练-测试RoPE外推差距。在长视频和短视频生成中，Echo-Infinity取得了最先进的性能，并且据我们所知，首次展示了有前景的24小时（>130万帧）实时滚动生成，为无限视频生成指明了一条实用路径。

## Abstract
We present Echo Infinity, an autoregressive (AR) framework towards real-time infinite video generation that employs a learnable evolving memory to dynamically filter, abstract, and compress any-length history at constant cost. Existing methods mainly curate memory with predefined KV-cache schedules, fixed-ratio heuristic compression, or inference-time RoPE adaptation. These designs inevitably lose historical information and amplify compounding errors due to their limited cache window and ignorance of autoregressive generation noise. Inspired by human memory consolidation, Echo-Infinity replaces handcrafted memory curation with learnable Memory Query, which are updated by attention and a gating mechanism when past frames are evicted from the local window. The queries are optimized end-to-end with the video diffusion transformers (DiTs), forming an evolving memory that supports arbitrary compression ratios with constant computation independent of video length. They also act as a generalizable generation prior, improving quality even when only the optimized initial state is used. We further introduce Unified Relative RoPE Recipe, which anchors the sink frames to start from id 0 and lets the newest frame id grow at most to the DiTs' pretrained maximum temporal RoPE id throughout training and inference, freeing the model from the finite RoPE constraint and closing the train-test RoPE extrapolation gap. In long and short video generation, Echo-Infinity achieves state-of-the-art performance, and, to our knowledge, demonstrates promising 24-hour (>1.3 M frames) real-time rollouts for the first time, suggesting a practical path toward infinite video generation.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心挑战**：现有实时自回归视频生成方法（如基于 DMD 的流式方案）在长时生成中面临两大瓶颈：
  - **KV 缓存无限增长**：随着帧数增加，历史 Key-Value 缓存线性增长，导致显存爆炸和计算成本不可控。
  - **时间旋转位置编码（RoPE）外推失败**：自回归生成时，时间索引会超出模型预训练时的最大索引（如 Wan2.1 的 fmax=20），引发质量崩溃甚至数值溢出。
- **现有方案局限**：当前方法主要采用手工设计的记忆管理策略，包括窗口截断、手工 KV 缓存调度、启发式压缩等。这些方法缓存窗口有限、压缩比固定、忽视自回归噪声累积，导致历史信息丢失和误差放大，无法支持真正无限长的视频生成。
- **本文目标**：借鉴人类记忆巩固机制（工作记忆→长期记忆的过滤、抽象、压缩），实现端到端可学习的演化记忆，以恒定计算代价支持任意长度视频的实时生成。

## 2. 方法论：核心思想、关键技术细节、公式/算法流程

- **核心思想**：用一组可学习的**记忆查询（Memory Query）** 替换手工记忆管理。每当帧从局部窗口驱逐时，通过交叉注意力+门控残差更新记忆查询，实现过滤、抽象和压缩。同时引入**统一相对 RoPE 配方（Unified Relative RoPE Recipe）**，在训练和推理中保持所有活动时间 id 在预训练范围内。
- **关键技术细节**：
  1. **三层 KV 缓存组织**：模拟人类记忆层次
     - **沉降帧（Sink Frames）**：固定数量的持久全局锚点（NS=3）。
     - **局部窗口（Local Window）**：短期工作缓冲区（NW=9）。
     - **记忆查询（Memory Queries）**：可学习的演化长期记忆（NQ=3 个帧的 token，共 4680 个 token）。
  2. **记忆查询更新机制**（图 3(b)）：
     - 当局部窗口滑动，被驱逐帧的 KV 对（最后一层）进入编码器 Enc（2 层交叉注意力），编码器根据当前记忆查询 Q 提取相关信息得到最终 Q̃。
     - 使用 sigmoid 门控残差更新：g = σ([Q; Q̃]W_gate)，Q ← g ⊙ Q + (1-g) ⊙ Q̃。
     - 该门控控制新旧记忆的混合，类似人类记忆巩固中的选择性保留。
  3. **注入与端到端优化**：
     - 每层 DiT 中，记忆查询经线性投影生成 K_Q/V_Q，与沉降帧和局部 KV 拼接后参与因果注意力。
     - 记忆参数（Q, Enc, W_gate, W_K, W_V）与生成模型 G_θ 在 DMD 损失下联合优化。
     - 阶段一（5s 短训练）全参数优化；阶段二（60s 长 tuning）子片段边界梯度截断。
  4. **统一相对 RoPE 配方**（图 3(c)）：
     - **增长阶段**：当前帧结束 id 从 sink 大小逐步增长至 fmax。
     - **成熟阶段**：当前帧结束 id 固定在 fmax；每次生成新 chunk 时，所有非沉降帧 id 向后旋转（等价于旧帧前移），沉降帧始终 id=0。
     - 这样所有时间 id 始终在 [0, fmax] 内，消除训练-推理 RoPE 外推差距。
  5. **算法流程**（Algorithm 1）：
     - 每步生成 B=3 帧 chunk，首先计算各部分的 RoPE id 分配（式(3)）。
     - 执行三层注意力，得到新 chunk 的 KV 并追加到局部窗口。
     - 若发生驱逐，调用更新函数刷新记忆查询。
     - 所有 id 按相对 RoPE 重调度。

## 6. 主要结论与发现

- Echo-Infinity 在长视频（30s/240s）、短视频（5s）、交互视频（60s）三个任务上均达到 SOTA，尤其在用户偏好上大幅领先。
- 端到端学习的记忆查询能有效过滤、抽象和压缩历史信息，固定容量即可处理任意长度历史，远超固定压缩比方法。
- 统一相对 RoPE 配方彻底消除了训练-推理 RoPE 外推差距，避免质量下降和溢出。
- 首次展示了 24 小时（>130 万帧）实时视频生成，质量稳定、一致性良好，验证了无限视频生成的可行性。
- 记忆查询优化后可作为通用生成先验，即使禁用更新也能提升短视频质量。

## 7. 优点

- **方法创新性**：将人类记忆巩固机制引入视频生成，用可学习记忆查询替代手工调度，实现端到端自适应压缩。
- **技术完整性**：同时解决 KV 缓存无限增长和 RoPE 外推两大问题，方案统一且兼容现有 DiT 架构。
- **实验充分性**：覆盖多任务、多时长、多指标，含用户研究，对比基线最新且强。
- **效率与性能平衡**：仅增加 10.6% 吞吐开销，仍保持 18.5 FPS 实时生成。
- **可迁移性**：记忆查询作为先验可独立使用，无需更新即可提升短视生成。

## 8. 不足与局限

- **交互生成优化不足**：记忆查询和统一 RoPE 主要针对单提示长视频，交互多提示场景的优化（如快速适应新提示）尚未充分探索。
