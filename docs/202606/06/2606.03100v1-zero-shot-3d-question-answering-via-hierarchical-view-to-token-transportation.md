## 摘要
近年来，通过2D视觉-语言模型（VLM）进行零样本3D场景理解因其实用的空间推理能力而受到越来越多的研究关注。通常，从3D点云中采样多个2D视图，并输入预训练的VLM以回答给定问题。这种范式凸显了输入上下文质量的关键作用，并提出了在有限输入预算下尽可能保留与任务相关的3D细节的挑战。我们提出了\texttt{KeyVT}，一种在视图和令牌两个层次上进行输入上下文收集的分层方法。具体来说，我们将像素特征与相机参数相结合，并基于语义内容和几何位置评估视图重要性，从而得到空间一致且任务相关的视图。此外，我们通过在最优传输（OT）框架下识别代表性令牌来解决所选视图中各个块之间的冗余问题，其中视图令牌和关键令牌被表示为嵌入空间中的两个离散分布。这些关键令牌通过最小化OT距离来覆盖所有视图特征。我们在三个广泛使用的基准上评估了我们的框架，显示出相对于现有无调优方法的显著改进，以及与基于训练的方法相当的性能。

## Abstract
Recently, zero-shot 3D scene understanding via 2D Vision-Language Models (VLMs) has gained increasing research interest due to their promising spatial reasoning capabilities. Typically, multiple 2D views are sampled from a 3D point cloud and fed into pre-trained VLMs to answer a given question. This paradigm highlights the critical role of input context quality and raises the challenge of retaining as many task-relevant 3D details as possible under a limited input budget. We propose \texttt{KeyVT}, a hierarchical approach for input context collection at both the view and token levels. Specifically, we combine pixel features with camera parameters and assess view importance based on both semantic content and geometric position, resulting in spatially consistent and task-relevant views. Furthermore, we address redundancy among patches across selected views by identifying representative tokens under the optimal transport (OT) framework, where view tokens and key tokens are formulated as two discrete distributions in the embedding space. These key tokens are expected to cover all view features by minimizing the OT distance. We evaluate our framework on three widely used benchmarks, demonstrating significant improvements over existing tuning-free methods and performance comparable to training-based approaches.

- **核心挑战**：在有限的输入预算（如固定数量的视觉令牌）下，如何尽可能保留与任务相关的3D细节（如主要物体及其周围环境），同时避免冗余和噪声。
- **研究动机**：现有方法要么简单均匀采样导致噪声，要么只依赖查询-视图语义相似度而忽略几何信息，并且后续的令牌压缩方法（如聚类）对密集特征敏感，难以捕获多样化的令牌。
- **整体意义**：提出一种无需微调的分层输入收集方法，显著提升零样本3D问答性能，使2D VLM能够基于更丰富、更紧凑的3D上下文进行推理，性能接近甚至超越有训练的3D专用模型。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：采用“先选视图、再选令牌”的分层策略，在视图层级利用几何信息选择空间一致且任务相关的视图，在令牌层级通过最优传输（Optimal Transport, OT）从选中的视图中压缩出代表性令牌，从而在固定预算内保留最多信息。
- **关键技术细节**：
  - **KeyV（几何感知视图选择）**：
    - 利用每张视图的相机参数（位置t、旋转R）计算视图间距：距离D = ||C0 - Ci|| + θ(R0,Ri)，其中C为相机中心，θ为旋转角度差。
    - 将视图序列按窗口大小δ分割成子场景{L1,...,L|L|}，每个子场景包含空间邻近的视图。
    - 用BLIP2计算每个子场景与问题的相关性分数rl = max(Ol) + mean(Ol)，其中Ol为子场景内各视图与问题的余弦相似度。
    - 根据子场景重要性加权分配关键视图数：Nl = floor(K * Wl / ΣWl')，其中Wl = rl * sqrt(|Ll|)。最后在每个子场景内选择Top-Nl个相关视图，连接形成关键视图集M*。
  - **KeyT（OT引导的令牌选择）**：
    - 将关键视图的所有令牌嵌入表示为分布P = Σα_n * e_n，目标学习M个虚拟令牌Q = Σβ_m * c_m（M < N）。
    - 定义成本矩阵C_{n,m}=1-cos(e_n,c_m)，通过Sinkhorn算法最小化熵正则化OT距离：dC,γ(P,Q)=min_{T∈Uγ}⟨T,C⟩，其中Uγ为约束运输计划的集合。
    - 通过Adam优化器更新虚拟令牌Q（学习率1e-2，迭代10~15次）。
    - 为与VLM兼容，依据学习到的运输计划T，为每个虚拟令牌cm选择前SM个真实令牌（即运输概率最大的列对应的行令牌），拼接所有选定令牌作为最终输入I。
- **算法流程**：整体框架如图2所示：输入2D视图序列及问题→BLIP2计算语义相似度→KeyV进行子场景划分与按重要性分配视图→获得关键视图M*→提取令牌嵌入→KeyT初始化虚拟令牌并通过OT距离最小化更新→根据运输计划选取真实令牌→送入VLM生成答案。

## 6. 主要结论与发现
- KeyVT在几乎全部指标上优于现有无调优方法（AKS、DivPrune、FLoC），在某些指标上甚至超越基于训练的方法（如cdViews、Video-3D）。
- 几何感知的KeyV比纯语义的AKS更有效，尤其在需要空间一致性的3D场景中。

- 该方法对相机参数误差具有鲁棒性（5%噪声内性能稳定）。
- 整体表明：“提供充分的、与任务相关的3D视觉细节”是发挥2D VLM空间推理能力的关键。

## 7. 优点
- **无需训练**：完全无调优，仅需VLM推理前的一次轻量级优化，便于快速部署。
- **分层设计**：首次同时考虑视图级和令牌级优化，有效利用几何信息与特征分布。
- **OT理论应用**：将令牌压缩公式化为分布对齐问题，相比启发式聚类更理论化且能捕获全局结构。
- **鲁棒性**：对相机参数噪声不敏感，适用于估计参数（如VGGT）的场景。
- **通用性强**：在多个VLM骨干上均一致提升，且在小模型上同样有效。
- **可解释性**：运输计划提供了虚拟令牌与真实令牌的对应关系，易于可视化。

## 8. 不足与局限
- **实验覆盖**：仅在3个室内数据集上评估，未涉及室外或大规模场景；且主要聚焦QA任务，未验证其在3D captioning、visual grounding等任务上的泛化能力。
- **偏差风险**：依赖BLIP2计算语义相似度，可能引入预训练模型的固有偏差（如对颜色的偏好）；相机参数估计误差在极端噪声下（10%）性能下降明显。
- **应用限制**：VSI-Bench部分缺少现成相机参数，需额外使用VGGT估计，增加了处理步骤；对于高度杂乱的场景或模糊的空间关系，KeyVT仍可能失败（如附图6所示）。
- **比较公平性**：与训练方法的比较中，输入预算（如视图数）不完全对齐（如Video-3D使用32帧的结论优于KeyVT的8等效帧）；未与最新使用更多视图的零样本方法（如BOLT、MDP3）直接对比。
- **计算开销**：虽然无训练，但KeyT的OT迭代（10~15步）仍比DivPrune慢，在实时性要求高的场景中可能受限。
