## 摘要
键值（KV）缓存增长是自回归解码中的一个主要瓶颈，因为内存和带宽随上下文长度线性扩展。现有的KV驱逐方法通常依赖于静态启发式或代理分数，这难以有效跟踪未来令牌的效用，并且随着相关性的变化会导致脆弱的驱逐。为了解决这个问题，我们引入了KVpop，它通过直接监督保留或丢弃决策来学习固定预算的KV驱逐策略。该评分器针对一种新颖的未来注意力目标进行训练，该目标可以在不具体化密集注意力图的情况下高效计算。我们进一步引入了一种基于延迟记忆的评分器，这在学习型驱逐方法中是独一无二的，它延迟固定步数的评分以利用近期未来上下文。在AIME和HMMT数学推理任务中，KVpop在Qwen3-4B模型上以75%的KV缓存压缩率保留了98%的全注意力性能，在88%压缩率下保留了97%的性能，持续优于已有的驱逐基线。Qwen3-8B模型表现更强，达到了接近全教师模型的性能。这些结果表明，使用未来注意力信号监督驱逐可以在保持质量的同时降低内存成本。

## Abstract
Key-value (KV) cache growth is a major bottleneck in autoregressive decoding, as memory and bandwidth scale linearly with context length. Existing KV eviction methods often rely on static heuristics or proxy scores, which poorly track future token utility and cause brittle eviction as relevance shifts. To address this, we introduce KVpop, which learns a fixed-budget KV eviction policy by directly supervising the keep-or-drop decision. The scorer is trained against a novel future-attention target, computed efficiently without materializing dense attention maps. We further introduce a delayed memory-based scorer that, uniquely among learned eviction methods, defers scoring for a fixed number of steps to exploit near-future context. On AIME and HMMT mathematical reasoning, KVpop retains 98% of full-attention performance on Qwen3-4B at 75% KV cache compression and 97% at 88% compression, consistently outperforming established eviction baselines. Qwen3-8B shows even stronger results, reaching near-full teacher performance. These results show that supervising eviction with future-attention signals cuts memory costs while maintaining quality.

- **本文目标**：提出一种**学习式、固定预算的 KV 驱逐方法**，通过直接监督未来注意力信号（而非代理分数）来决定保留或丢弃令牌，从而实现内存压缩与生成质量的更好平衡。

### 2. 论文提出的方法论

- **核心思想**：KVpop 是一个稀疏注意力后训练适配（retrofit）方法。每个 KV 头固定保留三部分：sink 令牌（开头少量令牌）、受保护滑窗（最近 w 个令牌）以及经重要性评分器选出的 top-k 长程令牌。总预算 B = s + w + k，确保推理时每个头的 KV 缓存大小恒定。
- **关键技术细节**：
  - **未来注意力目标 (Future-Attention Target)**：定义令牌 t 的平均未来注意力质量（在离开保护窗后收到的注意力概率均值），并通过 log 和 max 聚合得到目标分数。该目标无需具体化密集注意力矩阵，而是利用一次**转置注意力传递**（transposed attention pass）高效计算，复用稀疏注意力核返回的对数归一化因子（LSE）。
  - **边界感知损失 (Boundary-Aware Retention Loss)**：在每次查询位置 q，只比较新可驱逐令牌 t_new 与教师截止令牌 t_bnd 的预测分数，使用 pairwise logistic loss 推动学生分数在教师决策边界上正确排序。损失仅关注关键比较，计算复杂度 O(1) 每采样位置。
  - **运行 top-k 稀疏注意力**：使用 Fenwick 树（binary indexed tree）在线维护可驱逐令牌的静态优先级排名，每个查询位置只需插入一个新令牌并检索第 k 高的排名阈值，避免了全量重排序，复杂度 O(S log S) 每个头。
  - **状态性评分器**：支持**延迟评分**（delayed scoring）。不同于在令牌插入时立即评分，KVpop 可使用 mLSTM 记忆单元在令牌离开保护窗时才评分，从而利用保护窗内累积的近期未来上下文（即该令牌后的数个生成步骤的信息）进行更准确的判断。评分器参数仅通过前述边界损失更新，不参与主模型反向传播。
- **公式与算法描述**（文字版）：
  - 目标分数 \( r_{h,t}^{tgt} = \log(\epsilon + \sum_{d = t+w}^{S-1} p^{(h,g)}_{d \to t} / N_t) \)，其中 p 为注意力概率，N_t 为归一化项。
  - 有效分数加入时间衰减：\( r_{h,t}(q) = r_{h,t} + \lfloor (q - t)/n \rfloor \log \gamma_h \)。
  - 损失函数：\( \mathcal{L} = \sum_{q,h} \omega_{q,h} \operatorname{softplus}\left( -y_{q,h} (\hat{r}_{h,t_{\text{new}}}(q) - \hat{r}_{h,t_{\text{bnd}}}(q)) / \tau \right) \)，其中 y=+1 表示教师保留，-1 表示丢弃。
  - 训练时总损失为 KL 蒸馏损失 + 上述边界损失。

### 7. 优点

- **监督信号直接**：首次将未来注意力质量作为监督目标，而非代理分数或可微分松弛，训练目标与最终驱逐决策高度一致。
- **高效实现**：未来注意力目标通过转置注意力线性时间计算，无需稠密注意力矩阵；运行 top-k 使用 Fenwick 树，避免每次查询全量排序。
- **创新延迟评分**：利用保护窗提供的“近未来上下文”进行评分，是学习式驱逐方法中首次引入，原理自然且效果显著。
- **边界损失设计精巧**：仅关注教师决策边界处的关键比较，计算开销低，且引导评分器聚焦于区分保留/丢弃的关键分界线。
- **跨域泛化**：仅用数学推理数据训练，即能在科学和代码任务上保持良好性能，说明学到的驱逐模式具有一定迁移能力。
- **推理效率**：同质化预算使得缓存管理简单，适合 GPU 内核高效执行，相比 DMS 有延迟优势。

### 8. 不足与局限

- **实验覆盖有限**：仅在 Qwen3-4B/8B 上验证，未测试其他主流 LLM（如 Llama-3、Mistral）或更大模型（如 70B+），泛化性存疑。
