## 摘要
分类法提供了概念间语义关系以及视觉与语言内在组织的关键信息。尽管大规模多模态模型（LMMs）能力惊人，但它们通常缺乏分类知识，导致分层视觉识别（HVR）一致性较低。这些模型通常仅依赖于微调时的语言建模目标，缺乏明确的分类感知正则化。为了解决这个问题，我们提出了分层表示正则化（HiR^2），一种简单即插即用的正则化器，可提高LMMs中的分层一致性。具体来说，我们引入了一个语义感知的视觉树构建框架，该框架从由文本线索引导的中间LLM层提取从粗到细的视觉特征。该正则化器结合了两个互补目标：一个分类蕴涵损失，通过Lorentz模型中的双曲蕴涵锥强制分层；以及一个判别性分散损失，在不干扰径向分层结构的情况下，促进单位球面上语义相似嵌入的角分离。大量实验表明，HiR^2有效地捕获了不同LMMs和微调方法中的分类结构。代码可在https://github.com/PKU-ICST-MIPL/HiR2_ICML2026获取。

## Abstract
Taxonomies provide key information about the semantic relationships between concepts and the inherent organization of vision and language. Despite their impressive capabilities, large multimodal models (LMMs) often lack taxonomic knowledge, leading to low hierarchical visual recognition (HVR) consistency. These models typically only rely on language modeling objectives during fine-tuning and lack explicit taxonomy-aware regularization. To address this, we propose Hierarchical Representation Regularization ($HiR^2$), a simple plug-and-play regularizer that improves hierarchical consistency in LMMs. Specifically, we introduce a semantic-aware visual tree construction framework that extracts coarse-to-fine visual features from intermediate LLM layers guided by textual cues. The regularizer combines two complementary objectives: a taxonomic entailment loss that enforces hierarchy via hyperbolic entailment cones in the Lorentz model, and a discriminative dispersive loss that promotes angular separation of semantically similar embeddings on the unit sphere without disturbing the radial hierarchical structure. Extensive experiments demonstrate that $HiR^2$ effectively captures taxonomic structures across diverse LMMs and fine-tuning methods. Code is available at https://github.com/PKU-ICST-MIPL/HiR2_ICML2026.

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- 在标准语言建模损失基础上，增加两个辅助损失，显式约束LMMs中间层的视觉表示遵循分类学树结构：
  - **语义感知视觉树构建**：从LLM最后层隐藏状态中提取不同层级（粗→细）的视觉特征。
  - **双曲蕴含损失**：在**Lorentz模型**（双曲空间）中强制父子概念间的几何包含关系。
  - **判别性分散损失**：在**单位球面**上增加兄弟类别之间的角分离度，提升细粒度判别性，同时不干扰双曲空间中的径向层次结构。

### 2.2 关键技术细节
1. **语义感知视觉树构建**：
   - 利用文本提示（如不同层级类别名称）作为查询，通过**非参数交叉注意力**机制聚合图像token隐藏状态，得到每个层级对应的视觉表示 \(v_i^E \in \mathbb{R}^D \)。
   - 文本特征按层级平均池化得到查询 \(q_i\)，计算与图像token \(h_j\) 的注意力权重，加权求和得到层级视觉特征。

2. **曲率感知缩放（Curvature-aware Scaling）**：
   - 在映射到双曲空间前，根据batch内嵌入的平均范数 \(\bar{r}\) 和目标曲率 \(\kappa\) 进行缩放，使嵌入半径与双曲尺度匹配，增强数值稳定性。

3. **双曲蕴含损失（Taxonomic Entailment Loss）**：
   - 使用**双曲蕴含锥**：对于父节点 \(p\) 和子节点 \(c\)，要求 \(c\) 位于 \(p\) 的锥形区域内。
   - 损失函数：\(\mathcal{L}_{\text{ent}}(p,c) = \max(0, \angle(p,c) - \omega(p))\)，其中 \(\omega(p)\) 为锥的半孔径。
   - 对所有相邻层级的父子对求和：\(\mathcal{L}_{\text{ent}} = \sum_{i=1}^{H-1} \mathcal{L}_{\text{ent}}(v_i^L, v_{i+1}^L)\)。

4. **判别性分散损失（Discriminative Dispersive Loss）**：
   - 将欧式视觉特征投影到**单位球面**（\(\hat{v}_i = v_i^E / \|v_i^E\|\)），仅保留角度信息。
   - 对同一父节点下的兄弟节点对，施加**边界基于角度的排斥损失**：\(\mathcal{L}_{\text{disp}}(S_p) = \sum_{i \neq j} \max(0, m - \arccos(\hat{v}_i^\top \hat{v}_j))\)。
   - 引入**父感知记忆库**（Parent-aware Memory Bank）维护历史嵌入，增强长期判别性。
   - 采用**线性调度激活**（从0逐渐增加到1），避免训练初期打乱语言建模。

5. **总损失**：\(\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{LM}} + \lambda_{\text{ent}} \mathcal{L}_{\text{ent}} + \lambda_{\text{disp}} \mathcal{L}_{\text{disp}}\)。

### 2.3 理论证明（附录A）
- 定理4.1：基于双曲测地距离或切空间角度的分散损失会引入径向梯度，干扰层次结构。
- 定理4.2：提出的球面角分散损失对双曲嵌入的径向分量梯度为0，不干扰层次。
- 定理4.3：增大球面角分离度会单调增加双曲空间中的角分离度，提升兄弟判别性。

## 3. 实验设计

### 3.1 数据集与任务
| 数据集 | 层次数 | 叶子节点数 | 划分方式 |
|--------|--------|-------------|----------|
| **iNaturalist-2021 (iNat21)** Plant | 6层 | 4,271 | 均匀分为 base / novel |
| **iNat21** Animal | 6层 | 5,388 | 均匀分为 base / novel |
| **CUB-200-2011** | 4层 | 200（鸟类常见名） | 100 base / 100 novel |

- 训练设置：**1-shot**（每个base类仅1张训练样本），训练在base类上，评估在base类（base-to-base）和novel类（base-to-novel）上。

### 3.2 对比方法
- **基模型**：Qwen2.5-VL-3B, Qwen2-VL-2B, Intern3.5-VL-1B, LLaVA-OV-1.5-4B（涵盖不同参数量级和架构）。
- **微调方法**：**SFT（监督微调）** 和 **DFT（动态微调）**。
- 对比基线：纯SFT/DFT（无正则化） vs SFT/DFT + HiR²。

### 3.3 评估指标
- **HCA**（Hierarchical Consistent Accuracy）：要求从根到叶全路径预测正确（最严格）。
- **POR**（Point-Overlap Ratio）：所有层级正确比例的平均。
- **S-POR**（Strict Point-Overlap Ratio）：最长连续正确段比例。
- **TOR**（Top Overlap Ratio）：相邻层级正确对的比例。
- **Acc_leaf**：叶子节点准确率（仅在下属消融实验中报告）。

### 3.4 实验设置补充
- 使用**四选一VQA**形式进行闭集评估，每个层级独立提问；干扰项通过与SigLIP计算相似度选择最接近的3个错误标签。
- 所有实验在**4×A6000 GPU**上完成，每个GPU batch size=1，梯度累积2步，训练10个epoch。

## 6. 主要结论与发现
1. **HiR² 可一致提升LMMs的层次视觉识别性能**：在所有数据集、模型、微调方法上，HCA、POR、S-POR、TOR均获得显著增益（如iNat-Animal base HCA提升+4.01%，CUB-200 base HCA提升+26.02%）。
2. **兼容性强**：不仅适用于SFT，也适用于DFT；不仅适用于Qwen系列，也适用于LLaVA-OV、InternVL。
3. **泛化能力强**：在base-to-novel评估中同样有效，说明正则化学到的层次结构能泛化到未见类别。
4. **两种损失互补**：纯Lent提升层次一致性，纯Ldisp提升兄弟判别性，二者联合达到最佳。
5. **球面角分散优于双曲/切空间变体**：理论证明和实验均支持其不干扰径向层次，且提升叶子节点准确率。
6. **最佳正则化层为最后一层**：过早层正则化可能破坏早期表示，最后一层作为与语言生成直接相关的表示层最有效。

## 8. 不足与局限
- **实验覆盖范围有限**：
  - 仅测试了1-shot设置，未探索更多训练样本（如full-shot）下的表现可能不同。

  - 模型规模集中在1B~4B，未验证更大模型（如7B、13B）上的有效性（可能受算力限制）。
- **应用限制**：
  - 需要预先知道分类学树结构，对于无标签层次结构的数据需要额外构建。
  - 正则化损失依赖于批次内同一父节点下的兄弟样本，小batch size下效果可能下降（文中使用batch size=1 + 梯度累积2步，可能限制记忆库效果）。
  - 未讨论计算开销定量分析（虽然声明“little computational overhead”，但缺少实际FLOPs或训练时间对比）。
- **潜在偏差风险**：

  - 四选一VQA的评估形式可能低估模型在开放式预测中的表现（与真实HVR场景有差距）。
- **未对比最新层次视觉识别方法**：仅与纯SFT/DFT基线对比，未与专门为层次分类设计的其他方法（如HiCLIP、Hyperbolic CLIP等）在同等条件下比较（但这些方法非LMMs，可能不直接可比）。
