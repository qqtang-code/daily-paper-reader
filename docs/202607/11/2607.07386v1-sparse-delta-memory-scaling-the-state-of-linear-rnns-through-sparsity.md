## 摘要
线性注意力模型允许固定的状态大小和每令牌固定的计算量。然而，由于其有限的状态大小，线性注意力模型在长上下文回忆方面落后于基于softmax注意力的Transformer架构。增加线性注意力的状态大小可以提高回忆性能，但代价是更高的FLOPs。在这项工作中，我们引入了稀疏Delta记忆（SDM），这是一种通过稀疏寻址方案将门控线性RNN的隐藏状态扩展到更高数量级容量的架构。SDM扩展了门控DeltaNet架构，用稀疏读写大型显式记忆替代了密集的键值外积。我们证明，在isoFLOP约束下且参数数量相同的情况下，更高的状态记忆容量显著提升了上下文学习和长上下文检索任务的性能。此外，通过学习SDM记忆的初始状态并将其用作参数化记忆，我们表明该模型在广泛常识和推理任务上进一步改进。

## Abstract
Linear attention models allow a fixed state size and a fixed amount of compute per token. However, due to their limited state size, linear attention models fall behind in long-context recall compared to softmax-attention-based transformer architectures. Increasing the state size of linear attention improves recall performance but at the cost of higher FLOPs. In this work, we introduce Sparse Delta Memory (SDM), an architecture that scales the hidden state of gated linear RNNs to orders of magnitude higher capacity using a sparse addressing scheme. SDM extends the Gated DeltaNet architecture by replacing the dense key-value outer product with sparse reads and writes to a large explicit memory. We show that, under an isoFLOP constraint and with an identical number of parameters, a higher state memory capacity significantly improves performance on in-context learning and long-context retrieval tasks. Moreover, by learning the initial state of the SDM memory and therefore using it as a parametric memory, we show that the model further improves on a wide range of common-knowledge and reasoning tasks.

### 1. 核心问题与整体含义
线性注意力模型（如Gated DeltaNet、Mamba2）通过固定大小的隐藏状态实现每Token常量计算，但状态大小极为有限（通常仅几百KB），导致在长上下文召回任务上远逊于softmax注意力Transformer。增大状态可提升召回，但密集更新的计算复杂度随状态线性增长。本文的核心问题：**如何在几乎不增加FLOPs的前提下，将线性RNN的隐状态大幅扩展，从而提升长程建模能力？**  
整体含义：提出一种稀疏寻址方案（Sparse Delta Memory, SDM），使门控线性RNN的隐状态容量提升三个数量级，同时保持等FLOPs，进而在长上下文召回和上下文学习上显著超越原有方法，甚至与Full Attention持平或更优。

### 2. 方法论
**核心思想**：将Gated DeltaNet（GDN）的密集键值外积更新替换为对大型显式记忆表（Memory Table）的稀疏读写。利用Product-Key Memory（PKM）的索引机制，以O(√N×d + W²)的代价从N个槽中选出W个写槽和R个读槽。

**关键技术细节**：
- **稀疏键选择**：将输入投影到两个半向量，通过外积得到N个槽的分数，取top-W/top-R。
- **门控Delta写**：对选中的写槽，先按遗忘门衰减，再按输入门和Delta规则更新。
- **稀疏读**：对选中的读槽，按查询权重加权求和。
- **连接GDN**：当W=R=dqk且状态槽数为dqk时，退化为GDN。
- **学习初始状态M0**：将M0作为可学习参数，存储预训练知识，不增加推理FLOPs。

**公式（文字说明）**：
- 写操作：每个写槽i：Mt[i] ← αt·M_{t-1}[i] + βt·k_t^(i)·(v_t - αt·M_{t-1}⊤k_t)
- 读操作：yt = Σ_{i∈I_rt} q_t^(i)·Mt[i]
- 遗忘门αt、输入门βt均为可学习的标量门控。

### 6. 主要结论与发现
- SDM在等FLOP下，训练损失始终低于GDN，且缩放定律预测8B下甚至低于FullAttn。
- 长上下文召回：SDM在RULER上平均准确率31.2%（1.4B）和50.2%（8B），远超GDN（20.0%和34.2%）。
- 短上下文推理：SDM在多个常识/推理任务上的平均准确率高于GDN，8B时超越FullAttn。
- 学习初始状态M0：对SDM显著提升（代码perplexity从0.845降至0.822），对GDN几乎无影响（因状态太小）。
- 状态大小越大，性能越好（消融实验：432MB优于108MB优于27MB优于GDN的0.2MB）。
- 稀疏读写具有自适应模式：读写分布随预算调整。

### 7. 优点
- **创新性强**：首次将PKM与门控Delta规则结合，实现等FLOP下状态规模巨大跃升。
- **实验规范**：严格isoFLOP设计，所有模型共享参数与FLOPs；缩放定律拟合度高(R²≈0.999)；消融全面。
- **开源**：代码已在GitHub释放，促进可复现性。
- **实用价值**：在长上下文场景（软件工程、研究助理）有直接潜力，且内存占用在极端长序列下低于FullAttn的KV缓存。

### 8. 不足与局限
- **效率瓶颈**：当前SDM内核的MFU远低于GDN（因HBM随机访问慢于SRAM），训练吞吐慢1.49倍，推理慢约10%（但仍比FullAttn快6倍）。
- **内存占用大**：SDM状态大小与模型参数可比，对于资源受限设备可能不友好；但文末指出在全注意力处理20万token时KV缓存已大于SDM状态。
- **应用限制**：仅在文本领域验证，未涉及其它模态（如视频、图像）。微调阶段使用了量化（fp8/int4）以降低内存快照，但可能引入误差（虽实验似乎无损）。
- **实现复杂性**：稀疏读写导致训练代码较复杂（需要两指针归并、分段前缀和等自定义CUDA核），工程门槛高。
- **评估偏差风险**：长上下文RULER任务中，某些子任务（如multikey_2）对所有固定状态模型都很困难，SDM优势微弱；且文中未提供所有RULER任务的详细长度分解图（仅有部分），需参考附录。
