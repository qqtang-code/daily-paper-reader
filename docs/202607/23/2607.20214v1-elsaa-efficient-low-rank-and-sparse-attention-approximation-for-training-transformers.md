## 摘要
二次的N×N注意力分数矩阵仍是扩展Transformer至更长输入长度的主要障碍。现有高效注意力方法通常通过施加稀疏性（使每个查询仅关注少量键）或使用低秩/内核草图（将全局交互压缩为低维表示）来缓解这一瓶颈。我们提出ELSAA，一种高效的低秩与稀疏注意力近似。重要的是，ELSAA并不分解Transformer中学习到的投影或输出矩阵为稀疏和低秩因子，而是在密集投影生成Q、K、V后，近似诱导出的注意力分数算子本身：稀疏分支捕获选定的高相似性交互，低秩分支总结弥散的全局交互。由于两个分支可在分母质量差异很大的支持上归一化，ELSAA引入了一个分母感知融合项，根据稀疏分支相对于低秩分支的估计注意力质量来缩放稀疏分支。这提供了一个实用框架，无需生成完整的二次分数矩阵即可构建低秩和稀疏注意力输出，旨在实现长上下文训练，同时保留尖锐的标记级交互和广泛的上下文混合。

## Abstract
The quadratic $N\times N$ attention score matrix remains a central obstacle to extending Transformers to longer input lengths. Existing efficient attention methods usually reduce this bottleneck by either imposing sparsity, so that each query attends to only a small subset of keys, or by using low-rank/kernel sketches, so that global interactions are compressed into a lower-dimensional representation. We propose \emph{ELSAA}, an efficient low-rank and sparse approximation of attention. Importantly, ELSAA does \emph{not} decompose the learned projection or output matrices of the Transformer into sparse and low-rank factors. Instead, after dense projections produce $Q,K,V$, ELSAA approximates the induced attention score operator itself: a sparse branch captures selected high-similarity interactions, while a low-rank branch summarizes diffuse global interactions. Since the two branches can be normalized over supports with very different denominator mass, ELSAA introduces a denominator-aware fusion term that scales the sparse branch according to its estimated attention mass relative to the low-rank branch. This gives a practical framework for constructing low-rank and sparse attention outputs without materializing the full quadratic score matrix, aiming to enable longer-context training while preserving both sharp token-level interactions and broad contextual mixing.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：标准Transformer中的自注意力机制需要对长度为N的序列计算N×N的注意力分数矩阵，其时间和内存复杂度为O(N²)，这成为处理长序列（如长文档、高分辨率图像）时的主要瓶颈。

- **现有方法的局限性**：
  - 稀疏注意力方法（如局部窗口、LSH、路由等）能捕获高置信度的尖峰交互，但可能遗漏微弱但全局重要的交互。
  - 低秩/核方法（如Linformer、Performer、RACE等）能压缩全局信息、实现线性复杂度，但会平滑掉尖锐的标记依赖关系。
  - 两者单独使用均存在缺陷，且简单的相加会导致分支间尺度不匹配和计数重叠。

- **研究动机**：提出一种联合稀疏与低秩的注意力近似方法，在不显式构造完整N×N矩阵的前提下，同时保留局部高相似度交互和全局扩散交互，从而支持更长的上下文训练。

## 2. 论文提出的方法论

- **核心思想**：不分解Transformer的投影权重矩阵（如W_Q、W_K、W_V、W_O），而是在Q、K、V经过密集投影生成后，对注意力算子本身进行稀疏-低秩联合近似。包括一个稀疏分支（捕获高相似度交互）和一个低秩分支（总结全局扩散交互），并通过分母感知融合项来平衡两个分支的归一化尺度。

- **关键技术细节**：
  - **稀疏分支（sortLSH）**：使用排序的局部敏感哈希（LSH）将查询和键哈希并排序，分组为等大小块，在每个块内精确计算注意力。这确保每个查询只与块内的键交互，避免全矩阵构造。
  - **低秩分支（RACE）**：使用可微哈希草图，将键和值软分配到桶中，积累桶统计量，查询从这些摘要中读取，产生全局上下文的低秩近似。线性复杂度。
  - **分母感知融合**：由于稀疏分支和低秩分支的归一化分母质量差异很大，引入缩放因子 \( m_{\text{sparse},i} = \frac{d_{\text{sparse},i}}{d_{\text{sparse},i} + \lambda_i d_{\text{lr},i} + \epsilon} \)，其中 \( d \) 是估计的分母（注意质量），\( \lambda_i \) 可固定或可学习。
  - **门的融合**：通过学习两个独立的门（g_sparse, g_lr），计算最终输出：\( O_i = g_{\text{sparse},i} m_{\text{sparse},i} O_{\text{sparse},i} + g_{\text{lr},i} O_{\text{lr},i} \)。
  - **因果扩展**：通过递归分治和分块前缀和，将稀疏和低秩分支均扩展至因果（自回归）场景。

- **算法流程**（文字说明）：
  1. 输入序列X，经密集投影得Q、K、V。
  2. 运行低秩分支（RACE）得到O_lr和分母d_lr。
  3. 运行稀疏分支（sortLSH）得到O_sparse和分母d_sparse。
  4. 计算分母感知缩放因子m_sparse。
  5. 计算token级门控(g_sparse, g_lr)。
  6. 融合输出：O_head = g_sparse * m_sparse * O_sparse + g_lr * O_lr。
  7. 经输出投影W_O得到最终结果。

- **复杂度分析**：ELSAA的注意力交互次数为O(N(s + L_s2^γ))，其中s为稀疏块大小，L_s为哈希表数量，γ为哈希位数。典型设置下（N=32000, s=256, L_s=4, γ=4），交互次数约为全注意力的1%（减少约99%）。

## 6. 论文的主要结论与发现

- **ELSAA在多数长上下文任务上优于或持平高效注意力基线**：特别是在结构化视觉任务（Oxford-IIIT Pet, Flowers-102, Food-101）上比RACE提升4-7个百分点；在Text Retrieval @ 64K上精确注意力崩溃至50%随机，而ELSAA达65.34%，RACE为66.30%（说明近似注意力可作为隐式结构先验）。
- **分母感知融合项（m_sparse）是关键**：对比Sort Lsh RACE，ELSAA平均测试准确率提升1.33个百分点，证明该修正可有效校正分支尺度不匹配。
- **稀疏和低秩分支在不同数据模态中优势互补**：视觉任务受益更多于稀疏分支（局部高相似度主导），长文本任务（ArXiv分类）低秩分支更优（全局扩散证据聚合），ELSAA能够自适应平衡。
- **NIAH外推结果突出**：ELSAA在短于训练长度（512）时仍保持100%准确率（LSH失效），在长于训练长度（32K~65K）时远优于RACE（RACE崩溃），且接近Sparse LSH，同时避免了精确注意力的OOM问题。
