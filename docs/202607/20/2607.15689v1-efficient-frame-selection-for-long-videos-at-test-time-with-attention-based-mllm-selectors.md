## 摘要
使用多模态大语言模型（MLLM）理解长视频需要从数千个候选帧中选择紧凑的一组帧，然而识别正确的帧似乎首先需要理解视频。我们通过一个简单的观察解决了这一循环依赖问题：MLLM中经过验证选择的提取层的跨模态注意力已经提供了查询相关的帧证据，而无需自回归生成。我们利用这一特性构建了DAFS（基于动态注意力的预算感知帧选择），一种无需训练的帧选择器。即使只有2B参数的轻量级MLLM选择器，也可以通过将选定层的注意力通过查询条件聚合转换为相关性分数来提取帧级证据。这使得无需自回归解码即可进行跨帧比较。为了处理选择器自身的上下文约束，我们将候选池大小和每帧令牌预算的联合分配构建为一个离散优化问题，并通过动态规划求解。在32帧预算下，我们的选择器在Video-MME上比均匀采样提高了最多6.4个点，并在匹配帧预算下优于先前基于训练的选择器，同时无需重新训练即可在多个选择器和回答器骨干网络以及任务中泛化。

## Abstract
Understanding long videos with multimodal large language models (MLLMs) requires selecting a compact set of frames from thousands of candidates, yet identifying the right frames seemingly requires understanding the video first. We resolve this circular dependency with a simple observation: cross-modal attention at validation-selected extraction layers in MLLMs already provides query-relevant frame evidence without requiring autoregressive generation. We exploit this property to build DAFS (Dynamic Attention-based Budget-aware Frame Selection), a training-free frame selector. A lightweight MLLM selector, even with only 2B parameters, can extract frame-level evidence by converting selected-layer attention into relevance scores through query-conditioned aggregation. This enables cross-frame comparison without autoregressive decoding. To handle the selector's own context constraint, we formulate the joint allocation of candidate pool size and per-frame token budget as a discrete optimization problem solved by dynamic programming. Under a 32-frame budget, our selector improves over uniform sampling by up to 6.4 points on Video-MME and outperforms prior training-based selectors under matched frame budgets, while generalizing across selector and answerer backbones, and across tasks, without retraining.

### 方法论：核心思想、关键技术细节、算法流程

- **核心思想**：利用预训练 MLLM 中间层（验证选定）的**跨模态注意力**作为帧相关性信号——注意力权重已天然集中于查询相关的视觉令牌，无需生成答案。
- **关键技术**：
  - **选定层注意力提取**：对固定的小 MLLM（如 InternVL3.5-2B），在指定层 ℓ⋆ 提取文本到视觉的注意力矩阵 A ∈ ℝ^(T × (F·P))。
  - **查询条件聚合**：对每个帧的所有视觉令牌求和得到令牌级得分；再对重要查询令牌（如多选题选项 token）做**最大池化**，得到帧级显著性分数 ϕ_j。
  - **候选帧采样与空间压缩**：视频以 1 FPS 提取候选帧，再均匀采样 F(d) 帧；每帧通过非重叠平均池化压缩为 P(d) 个视觉令牌（选自 {16,25,36,49,64,100,144}）。
  - **动态规划预算分配**：将 F(d) 和 P(d) 的联合选择建模为离散优化，目标最大化验证集准确率，约束：F·P ≤ B、F 非递减、P 非递增。用 DP 求解最优全局策略。
- **算法流程**：
  1. 按视频时长 d 确定候选帧数 F(d) 和每帧令牌数 P(d)（查 DP 表）。
  2. 均匀采样 F(d) 帧，压缩为每帧 P(d) 令牌，拼接后输入小 MLLM 选择器。
  3. 从选定层提取注意力，通过重要 token 最大池化得到帧显著性分数。
  4. 排序选出 top-K 帧（K ≤ 最终回答器预算），原始分辨率送入回答器 MLLM。

### 主要结论与发现

- DAFS 在 32 帧预算下，Video-MME 上比均匀采样最高提升 **6.4 点**，且**超越所有先前基于训练的帧选择方法**（如 ViaRL、K-Frames、GIFT 等）。
- 选择器可泛化至不同回答器（提升 2~12 点）和不同任务（MMBench-Video 最优）。
- 推理成本远低于训练基方法（端到端 6.81s vs ViaRL 12.49s）。
- 注意力信号的有效性：中间层（如 L12-L15）注意力最具信息量。

### 优点

- **无需训练**：直接利用预训练 MLLM 的注意力，节省训练资源和数据标注。
- **轻量高效**：小选择器（2B 参数）单次前向即可完成帧评分，无需自回归。
- **动态预算优化**：DP 联合分配候选帧数和令牌压缩，适应不同视频时长，平衡时空覆盖。
- **强泛化性**：即插即用至不同回答器、不同任务，无需重新训练或适配。
- **公平比较**：严格控制所有对比方法在相同答案器和帧预算下。

### 不足与局限

- **对验证集的依赖**：层选择和 DP 分配策略需在独立验证集上校准，可能存在验证集过拟合风险（尽管文中使用 OOD 数据）。
- **注意力信号质量受限于选择器规模**：小 MLLM 的注意力可能不够准确，尤其对复杂语义或长距离依赖。
- **未覆盖极长时间视频**：实验视频最长约 2 小时（MLVU），更长时间（如电影）的效果未知。
- **未与视频专用 MLLM 直接竞争**：文中提到 Keye-VL-1.5 在 64 帧下优于本方法（73.0 vs 67.5），但本文使用 32 帧预算，更严格的资源约束下表现已很强；但若资源允许更多帧，差距可能缩小。
- **未讨论“重要令牌”自动提取的鲁棒性**：对于开放 QA 需依赖 LLM 提取实体 token，该过程可能引入噪声。
- **仅单次前向**：无法像迭代方法（如 A.I.R.）逐步优化选择，在极端稀疏事件中可能遗漏。
